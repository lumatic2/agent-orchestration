> 2026.02 ~ 03 | 전략·재무·GTM·운영

## 전략

**2026 전략 방향 확정**

AI 도구의 상품화(commoditization)가 가속화되고 있다. OpenAI·Google이 저가 범용 도구를 공급하면서 기술 스펙 경쟁은 소모전이 된다. 이에 대한 답으로 Veeva Systems의 플레이북을 참조했다 — 도메인 특화 생태계를 구축해 범용 경쟁사가 침투하지 못하는 구간을 만드는 전략.

- **핵심 축**: Custom Engine Year — 커스텀 AI 엔진 수주·납품 역량 최대화
- **차별화 포지션**: "더 좋은 AI"가 아닌 "더 잘 운영되는 AI 생태계"
- **확정 전략문구**: *"플랜바이는 건축 AI를 단품으로 팔지 않습니다. 툴·워크플로우·교육이 결합된 통합 생태계로 장기 수익성과 재구매율을 높이는 기업입니다."*

이 문구가 회사소개서·홈페이지·영업 자료 전면 재작성의 기준점이 된다.

**딜 수주 판단 기준 설계 (`/decide-deal`)**

확정된 전략에 따라 어떤 딜을 받아야 하는지 판단하는 기준을 설계하고 Claude Code 스킬로 구현했다. 팀원 누구든 딜 정보를 입력하면 수주 여부를 즉시 판단할 수 있다.

- Fail-fast 2개 기준 (도메인·기간)
- 가중 채점 5개 기준 (계약 규모·재발주·실행 가능성·레퍼런스·도메인 적합성)
- 판정 기준: 26점 이상 → 적극 수주 / 20~25점 → 조건부 / 20점 미만 → 거절

## 재무

**성장 궤적**

| 연도 | 매출 | YoY |
|---|---|---|
| 2024 | 44,121,454원 | — |
| 2025 | 289,086,455원 | **+554% (6.5배)** |

매출이 +554% 성장하는 동안 영업비용은 +143% 증가에 그쳤다. 비용보다 매출이 4배 빠르게 성장하는 레버리지 구조가 데이터로 증명된다.

**2025년 말 기준 핵심 수치**

| 지표 | 수치 |
|---|---|
| 현금 잔고 | 4.2억원 |
| 런웨이 | 12~13개월 |
| Burn Multiple | 1.64x (합격선 1.5x 소폭 상회) |
| 재발주율 | 66.7% |
| Win rate | ~63% |

**2025 실질 손익 구조**

```
매출                       289,086,455원
COGS (인건비 + 클라우드)  -370,866,089원
────────────────────────────────────────
Gross Profit               -81,779,634원   (GM -28%)

SG&A (간접 인건비 + 임차·광고·운영비)
                           -532,628,487원
────────────────────────────────────────
영업이익                   -614,408,121원  (-6.1억)
```

현재 회계상 매출원가가 0원이었다. 계정별 원장을 직접 분석해 클라우드·AI API 비용과 납품 직접 인력 인건비를 COGS로 재분류했다. 2025년은 의도된 팀 투자 해였다.

**2026 레버리지 시나리오 (팀 현상유지 가정)**

| 매출 목표 | Gross Margin | 영업이익 | 비고 |
|---|---|---|---|
| 20억 (DIPS 목표) | **75%** | **+9.6억 흑자** | 밸류에이션 200~400억 |
| 10억 (손익분기점) | 49% | ≈ 0 | 현 파이프라인 달성 시 가능 구간 |
| 6억 (보수적) | 27% | -4.1억 | CNP동양급 딜 2건 + 재발주 기준 |

같은 팀으로 매출만 올리면 GM과 영업이익이 동시에 역전된다.

**Series A IR 준비 상태**

| 투자자 질문 | 준비 상태 | 필요 액션 |
|---|---|---|
| "YoY 성장률은?" | ✅ 답 가능 — +554% | — |
| "BEP는 언제?" | ✅ 답 가능 — 매출 10억 = BEP | 2026 예측 시나리오 정교화 |
| "Gross Margin은?" | ✅ 설계됨 — 실질 GM -28% → 매출 확대 시 역전 | 회계사와 COGS 재분류 적용 |
| "매출 집중도 리스크는?" | ⚠️ 취약 — CNP동양 47%, 삼성E&A 22% | 2026 H1 파이프라인 다변화 |
| "H1 매출 공백은?" | ⚠️ 취약 — 2025 H1 4.6M, H2 284.5M | 2026 H1 조기 클로즈 딜 확보 |
| "Cap table 구성은?" | ⚠️ 미완 — 주주명부 미수령 | 대표에게 주주명부 수령 요청 |

착수 타이밍: Phase 1(3~4월 내부 준비) → Phase 2(5~6월 네트워크 접촉) → Phase 3(7~8월 본격 IR).

**재무·세무 운영 매뉴얼**

담당자가 바뀌어도 즉시 운영 가능한 회계·재무·세무 운영 매뉴얼을 완성했다. 재무·회계·세무 전문가 AI를 설계하고 Slack 봇으로 연결해 팀 누구든 전문가 수준의 분석을 즉시 얻을 수 있는 구조다.
