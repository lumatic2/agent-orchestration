# Plana SaaS 역할 결정 제안
> 작성일: 2026-03-13

---

## 결론 먼저

**Plana의 전략적 역할 = Enterprise 인바운드 리드 채널 (Option B)**
SaaS 구독 매출(Option A)은 유지하되 최대화 목표는 두지 않는다.

---

## 근거

### 현재 매출 구조
- Won 매출 **94%가 커스텀 모델** (건설·부동산 섹터)
- SaaS 고객 = Tier 3-4 (소형 인테리어·설계사) → 구독 ARR 규모 제한적
- Enterprise (삼성 E&A, CBRE, Nikon-Trimble) = 전부 커스텀 계약, SaaS 미사용

### 현재 Plana 퍼널의 문제
지금 퍼널은 **SMB 구독 전환에만 최적화**돼 있음.
"무료 크레딧 → 유료 구독 업그레이드" 버튼만 있고,
Enterprise 유저가 커스텀 계약으로 넘어오는 경로가 없음.

### Series A 관점
- 런웨이 ~7.6개월, 투자 프로세스 6~12개월 → 지금 착수 필요
- IR 스토리: "Plana = 리드 채널 → 커스텀 모델 파이프라인" = Veeva 플레이북과 동일 구조
- SaaS ARR만 강조하면 현재 고객 믹스 설명이 어려움

## 지금 바꿔야 할 것 (한 가지)

Plana 제품 내에 **"커스텀 모델 문의"** 진입점 추가.

| 현재 | 변경 후 |
|---|---|
| 유료 구독 업그레이드 CTA만 존재 | 구독 CTA + **"대용량/전사 도입 문의하기"** CTA 병행 |
| 모든 유저 동일 퍼널 | Enterprise 계정 감지 (회사 이메일 도메인, 사용량) → 영업 알림 |
