## 필수 정보

| 항목 | 내용 |
|---|---|
| **프로젝트명** | (파일명으로 사용 — 공백 없이) |
| **유형** | 소설 / 책 / 논문 |
| **주제/설명** | (1-3문장) |
| **챕터 수** | (원하는 챕터 수, 없으면 AI 자동 결정) |
| **독자 대상** | |
| **언어** | 한국어 / English |

## 실행

```
/content
```

content-automation 파이프라인 현황 조회 및 관리.

## 결과물 위치
`~/Desktop/content-projects/[프로젝트명]/`
- `outline.md` — 목차
- `chapters/ch01.md`, `ch02.md`, ... — 각 챕터
- `compiled.md` — 최종 합본
