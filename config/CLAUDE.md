# 글로벌 규칙

- 한국어로 소통
- 간결하게 응답

## 모델/Effort 가이드

질문의 복잡도를 판단하여 현재 설정이 부적절하면 추천해라:

- **Sonnet + low~medium**: 파일 조회/편집, 간단한 스크립트, 정보 조회
- **Sonnet + medium~high**: 버그 수정, 기능 구현, 코드 리뷰
- **Opus + high**: 아키텍처 설계, 시스템 점검, 복합 분석, 전략 기획

현재 모델이 부적절하면 세션 시작 시 한 번만 안내:
"이 작업은 [모델]이 적합해요. `/model [모델]`로 바꾸시겠어요?"
