# Google Stitch (구 포멜리/Galileo AI) 가이드북

> Google Labs에서 무료 제공 중인 AI 네이티브 UI 디자인 도구.
> 말 한 마디로 UI 디자인 + 코드까지 생성.

---

## 정체

- **현재 이름**: Google Stitch
- **이전 이름**: Galileo AI (2025년 5월 구글이 인수 → Google Labs 통합)
- **개발사**: Google
- **접근**: Google Labs 계정으로 무료 접속 가능

---

## 핵심 기능

| 기능 | 설명 |
|---|---|
| Prompt-to-UI | 텍스트 설명만으로 완성된 UI 생성 |
| Sketch-to-UI | 손 스케치 / 스크린샷 → 고해상도 디지털 디자인 |
| 코드 추출 | HTML, CSS, React, Vue, Tailwind 코드로 내보내기 |
| Figma 연동 | 레이어/오토 레이아웃 유지한 채 Figma로 내보내기 |
| 멀티 플랫폼 | 웹 + 모바일 버전 동시 생성 |

---

## 다른 툴과 비교

| 구분 | Google Stitch | Figma | Canva |

| AI 역할 | 디자인 전체를 처음부터 생성 | 보조 기능 | 템플릿 기반 보조 |
| 강점 | 아이디어 → 코드 (속도) | 정밀 제어 + 디자인 시스템 | 방대한 템플릿 |

---

## 활용 시나리오

1. **MVP 프로토타이핑**: 디자이너 없이 개발자/기획자가 아이디어를 UI + 코드로 즉시 검증
2. **디자인 시안 탐색**: 빈 화면에서 고민 없이 여러 스타일 순식간에 생성
3. **디자인-개발 협업**: 디자인 → 코드 직접 변환으로 협업 비용 절감
4. **화이트보드 디지털화**: 회의 중 스케치를 즉시 앱 화면으로 변환

---

## 추천 사용 흐름 (투자봇 / 포트폴리오 적용)

```
1. 프롬프트 입력: "투자봇 대시보드 - 다크 테마, 수익률 차트, 전략별 현황"
2. Stitch가 UI 생성
3. Figma로 내보내기 → 세부 수정
4. 코드 추출 → Next.js 포트폴리오에 적용
```
