# [KNOWLEDGE] 국제조세 핵심 규정 및 수치 기준

## 이전가격 문서화 의무 (국조법 §11~16)

### 문서화 종류별 의무 기준

| 문서 종류 | 제출 의무 기준 | 제출 기한 |
|---|---|---|
| **마스터파일** | 직전 사업연도 연결재무제표 매출 1,000억원 이상 **AND** 국외특수관계인 거래 합계 500억원 이상 | 법인세 신고기한 후 12개월 이내 |
| **로컬파일** | 국외특수관계인 거래 합계 50억원 이상 (무형자산 거래 10억원 이상 별도) | 법인세 신고기한 |
| **국가별보고서(CbCR)** | 다국적기업그룹 연결매출 1조원 이상 (7.5억 유로 기준) | 사업연도 종료 후 12개월 이내 |

> ⚠️ 문서화 의무 위반 시 가산세: 미제출·거짓 제출 시 거래금액의 0.5% (최대 3억원)

### 이전가격 방법론 선택 우선순위

| 방법 | 약칭 | 적합 거래유형 | 신뢰성 |
|---|---|---|---|
| 비교가능 제3자 가격법 | CUP | 원자재·상품, 금리 거래 | 최우선 |
| 재판매가격법 | RPM | 유통·판매 법인 (단순기능) | 2순위 |
| 원가가산법 | CPM | 제조·용역 법인 (단순기능) | 2순위 |
| 거래순이익률법 | TNMM | 복합거래, 비교가능 사례 부족 | 실무 多 |
| 이익분할법 | PSM | 고유 무형자산, 복잡한 통합 거래 | 마지막 수단 |

## 글로벌 최저한세 (Pillar Two, GloBE) — 국조법 §63~92

### 적용 대상

| 요건 | 기준 |
|---|---|
| 연결매출 | 직전 4개 사업연도 중 2개 이상 → 7.5억 유로(약 1조원) 이상 |
| 적용 세율 기준 | 국가별 실효세율(GloBE ETR) < 15% 시 추가세 납부 |
| 한국 시행 | 2024년 1월 1일 이후 개시 사업연도 (국조법 개정) |

### GloBE 세금 유형

| 구분 | 설명 | 우선순위 |
|---|---|---|
| **QDMTT** (적격국내최저한세) | 한국 내 저세율 법인에 대해 한국이 직접 추가세 과세 | 최우선 (IIR 대체) |
| **IIR** (소득산입규칙) | 모회사 거주지국에서 저세율 자회사 소득에 추가세 부과 | 2순위 |
| **UTPR** (과세미달이익규칙) | IIR 미적용 시 다른 그룹 구성원 국가에서 보충 과세 | 3순위 |

### GloBE ETR 계산 개요

```
GloBE ETR = 조정세액 ÷ GloBE 순소득
추가세(Top-up Tax) = (15% - GloBE ETR) × 초과이익

실질기반 제외금액(SBIE):
  = (유형자산 장부가액 × 8%) + (급여 총액 × 10%)  ← 첫 10년 경과규정 적용
```

## 피지배외국법인(CFC) 규정 — 국조법 §17~27

### CFC 판정 기준

| 요건 | 기준 |
|---|---|
| 지분 요건 | 내국인(특수관계인 합산) 지분 50% 초과 |
| 저세율 요건 | 해당 국가 실효세율 **15% 미만** (한국 법인세율의 70% 미만) |
| 적용 소득 | 수동소득(배당·이자·사용료·임대 등) 우선 과세 |

### CFC 적용 제외 요건

- 실질적 사업 영위 (제조·판매·용역 등 능동소득 50% 이상)
- 거주지국 세율이 15% 이상인 경우
- 배당 소득을 실제로 수취한 경우 (이미 과세된 경우)

### 배당 간주 과세 계산

```
간주배당 = CFC 유보소득 × 내국인 지분율
과세 시점 = 해당 사업연도 종료일 다음 날부터 60일 이내 신고
```

## 해외금융계좌 신고 — 국조법 §52~62

| 항목 | 기준 |
|---|---|
| 신고 의무 기준 | 매월 말일 중 **어느 하루라도** 잔액 5억원 초과 |
| 신고 기한 | 매년 6월 (다음연도 6.1~6.30) |
| 신고 대상 | 거주자 및 내국법인 (내국법인은 10억원 기준 별도 적용) |
| 미신고 과태료 | 미신고 금액의 10~20% (고의적 탈루 시 형사처벌 병행) |
| 자진 신고 감경 | 최초 적발 전 자진신고 시 과태료 50% 감경 |

## 사전가격합의(APA) 절차 개요

| 구분 | 내용 |
|---|---|
| 단독 APA | 국세청과 납세자 간 합의 (상대국 세무당국 미참여) |
| 쌍방 APA | 국세청 + 상대국 세무당국 + 납세자 3자 협의 |
| 신청 기한 | 사업연도 개시 전 or 거래 발생 전 신청 권장 |
| 적용 기간 | 통상 3~5년 (갱신 가능) |
| 실무 효과 | 이전가격 조사 리스크 완전 제거 |
