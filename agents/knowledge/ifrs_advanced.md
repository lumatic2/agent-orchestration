# [KNOWLEDGE] K-IFRS 주요 기준서 심화 (Planby 재무분석용)

## IFRS 16 — 리스 (2019년 시행)

### 리스이용자 회계처리 (사용권자산·리스부채)
```
최초 인식:
  리스부채 = 미래 리스료의 현재가치 (내재이자율 or 증분차입이자율)
  사용권자산 = 리스부채 + 선급리스료 + 리스개설직접원가 - 리스인센티브

후속 측정:
  리스부채: 유효이자율법으로 상각
  사용권자산: 정액법 상각 (또는 리스기간에 걸쳐)
```

### 손익계산서 영향
- 리스비용(운용리스) → 감가상각비 + 이자비용으로 분리
- 영업이익(EBIT) 증가, EBITDA 증가 효과
- 부채비율 상승, 초기 비용 증가 (이자비용 front-loading)

### 단기리스·소액자산 면제
- 리스기간 12개월 이하 또는 기초자산 가치 5,000달러 이하 → 리스료 비용처리 가능

## IFRS 9 — 금융상품 (2018년 시행)

### 금융자산 분류 (사업모형 + 현금흐름 특성)
| 분류 | 측정 | 요건 |
|---|---|---|
| 상각후원가(AC) | 원가 | 원리금 수취 목적, SPPI 충족 |
| 기타포괄손익-공정가치(FVOCI) | 공정가치 | 매도+수취 혼합 목적, SPPI 충족 |
| 당기손익-공정가치(FVTPL) | 공정가치 | 단기매매 or 위 외 |
| 지분증권 FVOCI | 공정가치 | 최초 선택, 재분류 불가 |

### 기대신용손실(ECL) 모형
```
Stage 1: 신용위험 유의적 증가 없음 → 12개월 ECL
Stage 2: 신용위험 유의적 증가    → 전체기간 ECL
Stage 3: 신용손상 발생           → 전체기간 ECL + 이자 순액 계상
```

## IFRS 3 — 사업결합 (취득법)

### 취득법 4단계
```
1. 취득자 식별
2. 취득일 결정
3. 이전대가·피취득자 순자산 공정가치 측정
4. 영업권(or 염가매수차익) 인식

영업권 = 이전대가 + 비지배지분 - 식별가능 순자산 공정가치
```

### 영업권 후속처리
- K-IFRS: 상각 금지, 매년 손상검사 (CGU 단위)
- K-GAAP(중소기업): 20년 이내 상각 가능

### 취득관련원가
- K-IFRS: 당기비용 (이전대가에 포함 불가)
- 단계적 취득: 기존 보유 지분 공정가치 재측정 → 재측정손익 당기 인식

## IFRS 10 — 연결재무제표

### 지배력 3요소
```
지배력 = ① 피투자자에 대한 힘
        + ② 변동이익에 노출 또는 권리
        + ③ 힘을 이용해 이익에 영향을 미치는 능력
```

### 연결 범위 결정
- 의결권 50% 초과: 원칙적 지배력
- 50% 이하도 실질지배력 있으면 연결 (사실상 지배력)
- 특수목적기업(SPE): 실질 지배 시 연결

### 비지배지분(NCI)
- 종속기업 순자산 중 지배기업 지분 외 부분
- 연결자본에 포함 (부채 아님)
- 포괄손익계산서: 지배기업 귀속분 / NCI 귀속분 구분

## IFRS 13 — 공정가치 측정

### 공정가치 서열체계 (Fair Value Hierarchy)
| 수준 | 투입변수 | 예시 |
|---|---|---|
| Level 1 | 활성시장 공시가격 | 상장주식, 국채 |
| Level 2 | 관측가능 투입변수 | 유사자산 가격, 수익률곡선 |
| Level 3 | 관측불가 투입변수 | 비상장주식 DCF, 내부모형 |

### 최고최선 이용(HABU)
공정가치 = 시장참여자가 자산을 최고최선으로 이용할 때 받을 금액

## IFRS 1 — 한국채택국제회계기준 최초 채택

### 최초 채택일 처리
- 비교기간 포함 가장 이른 기간 초일에 IFRS 전환대차대조표 작성
- 대부분 소급적용 원칙
- 면제 조항 활용 가능 (사업결합, 종업원급여 등)

## K-IFRS vs 중소기업회계기준 주요 차이

| 항목 | K-IFRS | 중소기업회계기준 |
|---|---|---|
| 적용 대상 | 상장법인, 금융기관 등 | 비상장 중소기업 (선택) |
| 영업권 | 상각 금지 + 손상 | 20년 이내 상각 |
| 리스 (IFRS 16) | 사용권자산·리스부채 | 운용리스 비용처리 가능 |
| 금융상품 | IFRS 9 (ECL) | 단순화된 손상모형 |
| 연결 범위 | 실질지배력 | 의결권 기준 |
| 공정가치 | 전면 적용 | 역사적 원가 우선 |
| 공시 | 광범위 | 간소화 |
