# [KNOWLEDGE] 재무전략·기업가치 핵심 프레임워크

## DCF (Discounted Cash Flow) 기업가치평가

### 기본 공식
```
기업가치(EV) = Σ [FCF_t / (1+WACC)^t] + 잔여가치(Terminal Value)

잔여가치 = FCF_(n+1) / (WACC - g)
  g: 영구 성장률 (통상 GDP성장률 ≈ 2~3%)
```

### WACC (가중평균자본비용)
```
WACC = Ke × (E/V) + Kd × (1-t) × (D/V)

Ke (자기자본비용) = Rf + β × (Rm - Rf)   [CAPM]
  Rf: 무위험이자율 (국고채 3년물, 한국 ~3.5%)
  β: 체계적 위험 (업종 평균 비교)
  Rm-Rf: 시장 위험프리미엄 (한국 ~5~6%)

Kd (타인자본비용) = 차입이자율 × (1-법인세율)
```

### FCF (잉여현금흐름) 계산
```
FCF = EBIT × (1-t) + 감가상각 - CAPEX - ΔNWC

EBITDA → EBIT → NOPAT → FCF 순서로 전개
NWC (순운전자본) = 유동자산 - 유동부채 (현금·단기차입 제외)
```

## 상대가치평가 (Multiple)

### EV/EBITDA
```
EV = 시가총액 + 순차입금 (= 차입금 - 현금)
EV/EBITDA: 한국 상장사 평균 8~12배 (업종별 상이)
```

### PER / PBR / PSR
| 배수 | 산식 | 적합 업종 |
|---|---|---|
| PER | 주가 / EPS | 수익성 안정 기업 |
| PBR | 주가 / BPS | 금융·자산 집약 |
| PSR | 주가 / 매출액 | 적자 성장기 기업 |
| EV/EBITDA | EV / EBITDA | M&A, 부채 고려 |

## OKR (Objectives & Key Results)

### 구조
```
Objective: 정성적·영감적 목표 (What)
  └─ KR1: 측정 가능한 결과 (How / 0~1.0 점수)
  └─ KR2: ...
  └─ KR3: ...
```

### 설계 원칙
- Objective: 3~5개 / 분기, 동사로 시작 ("달성하다", "구축하다")
- Key Result: Objective당 2~5개, 숫자로 측정
- 목표 난이도: 0.6~0.7 달성이 이상적 (1.0이면 목표가 낮음)
- 재무 KPI를 KR에 직접 연결: ARR, CAC, LTV, NRR

### 재무 KPI → KR 변환 예시
| 재무 지표 | KR 표현 |
|---|---|
| 매출 성장 | "ARR을 5억→8억으로 증가" |
| 수익성 | "영업이익률 -5%→+3%로 개선" |
| 자본효율 | "ROE 12% 달성" |
| 현금흐름 | "FCF 흑자 전환 (월 +3,000만원)" |

## BSC (Balanced Scorecard) 4관점

| 관점 | 핵심 질문 | 지표 예시 |
|---|---|---|
| 재무 | 주주에게 어떻게 보일까? | ROE, EPS, EBITDA |
| 고객 | 고객에게 어떻게 보일까? | NPS, Churn, CAC |
| 내부프로세스 | 어디서 탁월해야 하나? | 리드타임, 불량률 |
| 학습·성장 | 어떻게 개선·혁신하나? | 직원만족도, R&D투자 |

전략맵: 학습·성장 → 내부프로세스 → 고객 → 재무 (인과관계)

## ROI / 투자분석 지표

### 기본 수익률 지표
```
ROI   = (순이익 / 투자비용) × 100%
ROE   = 당기순이익 / 평균자기자본
ROA   = 당기순이익 / 평균총자산
ROIC  = NOPAT / 투자자본 (= 자기자본 + 이자부부채)
```

### EVA (Economic Value Added)
```
EVA = NOPAT - (WACC × 투자자본)
EVA > 0: 주주가치 창출
EVA < 0: 자본비용 미달, 가치 파괴
```

### 투자결정 기준
| 지표 | 채택 기준 | 장단점 |
|---|---|---|
| NPV | > 0 | 절대 가치, 가장 신뢰 |
| IRR | > WACC | 상대 수익률, 복수해 문제 |
| MIRR | > WACC | IRR 보완 (재투자율 가정 현실화) |
| Payback | 목표 회수기간 이내 | 유동성 중시, 시간가치 무시 |

## 자본구조 최적화

### MM 이론 (Modigliani-Miller)
- 완전자본시장: 자본구조 무관 (부채 = 자기자본)
- 법인세 있는 경우: 부채의 절세효과(tax shield) = Kd × t × D
  → 부채 ↑ = 기업가치 ↑ (단, 파산위험 고려)

### 트레이드오프 이론
```
최적 부채비율 = 절세효과 = 재무적 곤경비용
```

### 한국 실무 기준
- 부채비율 200% 이하: 재무건전성 통상 기준
- 이자보상배율 1.5배 이상: 금융기관 대출 기준
- DSCR (부채상환능력비율) = EBITDA / (원금상환 + 이자)

## 재무제표 분석 핵심 비율

### 수익성
| 지표 | 산식 | 해석 |
|---|---|---|
| 매출총이익률 | 매출총이익 / 매출액 | 핵심 제품 경쟁력 |
| 영업이익률 | 영업이익 / 매출액 | 영업 효율성 |
| EBITDA 마진 | EBITDA / 매출액 | 현금창출 능력 |
| 순이익률 | 당기순이익 / 매출액 | 최종 수익성 |

### 효율성
- 매출채권회전율 = 매출액 / 평균 매출채권
- 재고자산회전율 = 매출원가 / 평균 재고자산
- 총자산회전율 = 매출액 / 평균 총자산

### 유동성
- 유동비율 = 유동자산 / 유동부채 (200% 이상 건전)
- 당좌비율 = (유동자산 - 재고) / 유동부채 (100% 이상)

### 레버리지
- 부채비율 = 총부채 / 자기자본
- 이자보상배율 = EBIT / 이자비용
