## 노동법 핵심

### 근로계약 필수 기재사항 (근로기준법 제17조)
1. 임금 (구성항목·계산방법·지급방법)
2. 소정근로시간
3. 휴일
4. 연차유급휴가
5. 취업 장소·종사 업무
6. 취업규칙 교부 (상시 10인 이상)

### 해고 요건 (근로기준법 제23~27조)
**정당한 해고 요건:**
- 정당한 이유 필요 (일신상·행태·경영상 사유)
- 30일 전 예고 또는 30일분 통상임금 지급 (해고예고수당)
- **서면 통보 의무** — 이유·시기 명시 (미이행 시 부당해고)

**부당해고 판단 기준:**
- 사유 없는 해고, 절차 미이행, 불이익처분 (징계해고 포함)
- 구제 신청: 노동위원회 (해고일로부터 **3개월 이내**)
- 법원 소송: 해고 무효 확인 + 임금 상당액 청구

**해고 예외 (해고 불가 기간):**
- 업무상 재해 요양 중 + 그 후 30일
- 출산 전후 휴가 기간 + 그 후 30일

### 임금 관련
| 항목 | 기준 |
|---|---|
| 최저임금 | 2025년: 10,030원/시간 |
| 지급일 | 매월 1회 이상 정기 지급 |
| 임금체불 | 지급일 다음날부터 지연이자 연 20% |
| 퇴직금 | 1년 이상 계속근로 → 30일분 평균임금 × 근속연수 |
| 평균임금 산정 | 사유 발생 전 3개월 임금 총액 ÷ 근무일수 |

### 근로시간·휴가
| 항목 | 기준 |
|---|---|
| 법정 근로시간 | 1일 8시간, 주 40시간 |
| 연장근로 한도 | 주 12시간 (합의 필요) |
| 연장·야간·휴일 가산 | 50% 가산 |
| 연차휴가 | 1년 미만: 월 1일 / 1년 이상: 15일 (최대 25일) |
| 연차 사용촉진 | 미사용 수당 대신 사용 촉진 가능 (2년차부터) |

### 직장 내 괴롭힘 (근로기준법 제76조의2)
- 요건: 지위·관계 우위 + 업무상 적정범위 초과 + 신체적·정신적 고통·환경 악화
- 사용자 의무: 즉시 조사 → 행위자 징계 → 피해자 보호
- 신고처: 고용노동부 (국번없이 1350)

## 퇴직금 계산

```
퇴직금 = 평균임금 × 30일 × (근속일수 / 365)

평균임금 = (최근 3개월 임금 총액) / (최근 3개월 달력상 일수)
```

**퇴직연금 (DC형):** 매년 연간임금 1/12 이상 적립
**IRP 의무이전:** 퇴직금 300만원 초과 시 IRP로 지급 (2022.4~)

## 부당해고 구제 절차

```
해고 통보
    ↓
노동위원회 구제신청 (3개월 이내)
    ↓
조사·심문 (45일 내 판정)
    ↓
원직복직 명령 or 금전보상 명령
    ↓ (불복 시)
중앙노동위원회 재심 (10일 이내)
    ↓ (불복 시)
행정소송 (서울행정법원) — 15일 이내 제기
```

## 민사 손해배상

### 손해배상 요건 (민법 제750조)
1. 고의 또는 과실
2. 위법 행위
3. 손해 발생
4. 인과관계

### 손해의 종류
| 종류 | 내용 |
|---|---|
| 재산적 손해 | 적극적 손해 (실제 지출) + 소극적 손해 (상실 이익) |
| 정신적 손해 | 위자료 (금액은 법원 재량) |
| 특별손해 | 예견 가능했을 것을 요구 (민법 제393조) |

### 소멸시효 (중요!)
| 청구 유형 | 시효 |
|---|---|
| 일반 채권 | 10년 (민법) |
| 상행위 채권 | **5년** (상법) |
| 불법행위 손해배상 | 안 날로부터 **3년** / 행위일로부터 10년 |
| 임금 청구권 | **3년** (근로기준법) |
| 퇴직금 청구권 | **3년** |
| 산재 보험급여 | 3년 (산재보험법) |

> ⚠️ **시효 주의**: 청구 기한 임박 시 내용증명 발송 또는 소 제기로 시효 중단

## 내용증명 활용

**내용증명 발송이 필요한 상황:**
- 계약 해지 통보
- 임금·퇴직금 지급 청구
- 손해배상 청구 예고
- 소멸시효 중단 (시효 완성 임박)
- 채무불이행 최고

**효력:** 법적 강제력 없음, 단 발송 사실 증명 + 심리적 압박 + 시효 중단 효과

## 임금체불 대응 절차

```
임금체불 발생
    ↓
사업장 소재지 고용노동지청 진정 접수 (시효 3년 주의)
    ↓
근로감독관 조사 (통상 1~2개월)
    ↓
[확인 시] 사용자에게 시정지시 + 불이행 시 형사처벌 (3년 이하 징역 or 3천만원 이하 벌금)
    ↓
[미해결 시] 민사 소액심판 or 지급명령 신청
```

**소액심판:** 청구금액 3,000만원 이하, 1심에서 종결
**체당금 제도:** 도산 기업 근로자 → 근로복지공단이 체불 임금 선지급

## 스타트업 주요 계약 리스크

### 근로계약 주의사항
- 수습 기간 중 해고: 3개월 이내 수습은 해고예고 면제, 단 부당해고는 적용
- 경업금지 약정: 합리적 기간·범위·보상 없으면 무효 가능성 (통상 1~2년)
- 포괄임금제: 실 근로시간 측정 어려운 경우 한정, 오남용 시 무효

### SAFE / 전환사채 핵심 조항
| 조항 | 내용 |
|---|---|
| 전환가격 | 후속 투자 시 Cap 또는 Discount 기준 |
| MFN (최혜국 조항) | 후속 SAFE 더 유리하면 자동 적용 |
| Pro-rata right | 후속 투자 참여권 |
| 우선청산권 | 청산 시 원금 또는 배수 우선 회수 |

### 노동법 위반 리스크 Top 5 (스타트업)
1. 4대보험 미가입 (프리랜서 계약으로 위장)
2. 연차휴가 미지급·미사용 수당 누락
3. 야근 수당 미지급 (포괄임금제 오남용)
4. 해고예고 미이행
5. 퇴직금 분할지급 약정 (무효)
