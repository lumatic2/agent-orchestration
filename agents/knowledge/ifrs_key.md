# [KNOWLEDGE] K-IFRS 핵심 기준서 요약

## IFRS 15 — 고객과의 계약에서 생기는 수익

### 5단계 모형

```
1단계: 계약 식별
  → 상업적 실질, 권리·의무 식별 가능, 대가 회수 가능성

2단계: 수행의무 식별
  → 고객에게 이전하기로 약속한 재화·용역 중 구별되는 것
  → 구별 기준: ① 자체적으로 효익 향수 가능 ② 다른 약속과 별개로 식별 가능

3단계: 거래가격 산정
  → 변동대가(추정치·제약), 유의한 금융요소, 비현금 대가, 고객에게 지급할 대가

4단계: 거래가격 배분
  → 상대적 독립적 판매가격(SSP) 비율로 수행의무별 배분

5단계: 수익 인식
  → 수행의무를 이행할 때(한 시점 or 기간에 걸쳐)
```

### 기간에 걸쳐 인식 요건 (셋 중 하나)
- ① 고객이 기업 이행에 따라 동시에 효익 수취·소비
- ② 기업 이행이 고객 통제 자산 창출·가치 향상
- ③ 기업 이행이 대체 용도 없는 자산 창출 + 완료 부분 대가 청구권 존재

## IFRS 16 — 리스

### 리스 식별 (계약에 리스가 포함되었는지)
- 식별된 자산의 사용 통제권을 일정 기간 동안 이전하는 계약
- 공급자가 대체할 실질적 권리가 없어야 함

### 리스이용자 최초 측정

```
사용권자산 = 리스부채 + 선급리스료 + 리스개설직접원가 + 복구충당부채
리스부채 = 미지급 리스료의 현재가치 (증분차입이자율로 할인)
```

리스료 구성: 고정리스료 + 지수·요율 연동 변동리스료 + 잔존가치보증

### 면제 규정
- 단기리스 (리스기간 12개월 이하): 리스이용자 선택 → 정액 비용처리
- 소액자산 리스 (기초자산 USD 5,000 이하): 선택 → 정액 비용처리

## IFRS 9 — 금융상품

### 금융자산 분류

| 분류 | 조건 | 측정 |
|---|---|---|
| 상각후원가(AC) | 계약상 현금흐름 수취 목적 + SPPI 충족 | 유효이자율법 |
| FVTOCI (채무상품) | 수취·매도 목적 + SPPI 충족 | 공정가치, OCI |
| FVTOCI (지분상품) | 단기매매 외 지분상품, 선택 | 공정가치, OCI (재분류 불가) |
| FVTPL | 위 외 | 공정가치, 당기손익 |

SPPI: 원금과 원금 잔액에 대한 이자만 지급 (Solely Payments of Principal and Interest)

### 손상: 기대신용손실(ECL)

```
12개월 ECL: 신용위험 유의적 증가 없음
전체기간 ECL: 신용위험 유의적으로 증가

유의적 증가 기준:
- 연체 30일 초과 (반증 가능한 가정)
- 신용등급 하락
- 재무적 어려움 징후
```

## IFRS 3 — 사업결합 (취득법)

```
영업권 = 이전대가 + 비지배지분 + 기존보유지분 공정가치
       - 피취득자 식별가능 순자산 공정가치

이전대가: 교부주식 공정가치, 현금, 조건부대가(공정가치)
피취득자 순자산: 식별가능 자산·부채를 취득일 공정가치로 측정
```

## IFRS 10 — 연결재무제표

### 지배력 3요소
1. 피투자자에 대한 **힘** (의결권, 잠재적 의결권)
2. 피투자자에 대한 **관여로 변동이익에 노출**
3. 피투자자에 대한 힘을 이용해 **이익 금액에 영향**

### 주요 연결 조정
- 종속기업 투자주식 vs 피투자자 자본 상계
- 내부거래 제거 (재고, 유형자산, 대여금)
- 미실현손익 제거 (하향판매: 전액, 상향판매: 지분율)
- 비지배지분 계산

## IFRS 2 — 주식기준보상 (스톡옵션)

```
인식 금액 = 부여일 공정가치 × 귀속된 수량
각 보고기간 비용 = 누적 인식액 - 전기까지 인식액

공정가치 측정: Black-Scholes 또는 이항모형
파라미터: 주가, 행사가격, 기대변동성, 무위험이자율, 기대만기, 배당수익률
```

조건 변경:
- 유리한 변경: 추가 공정가치 차이를 잔여 귀속기간에 추가 인식
- 불리한 변경: 원래 조건 계속 적용

## IFRS 17 현황 (보험계약)

IFRS 17 (보험계약): 2023년 1월 적용 개시
- 측정 모형: BBA(일반모형), PAA(보험료배분접근법), VFA(가변수수료접근법)
- 계약서비스마진(CSM): 미실현이익을 서비스 기간에 걸쳐 인식

## 중요 실무 판단 기준 요약

| 이슈 | 기준 판단 포인트 |
|---|---|
| 수익 인식 시점 | 통제권 이전 시점 (IFRS 15) |
| 리스 여부 | 자산 사용 통제권 이전 여부 (IFRS 16) |
| 금융자산 분류 | 사업모형 + SPPI (IFRS 9) |
| 지배력 판단 | 힘·이익노출·연계 3요소 (IFRS 10) |
| 사업결합 | 취득법 적용, PPA (IFRS 3) |
| 손상 | CGU별 회수가능액 vs 장부금액 (IFRS 136) |
