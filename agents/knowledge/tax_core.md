# [KNOWLEDGE] 법인세 핵심 계산구조 및 세율

## 각사업연도소득 계산 구조

```
결산서상 당기순이익(손실)
  + 익금산입·손금불산입 (세무조정 가산)
  - 손금산입·익금불산입 (세무조정 차감)
= 차가감소득금액
  - 이월결손금 공제 (한도 있음)
  - 비과세소득
  - 소득공제
= 과세표준
  × 세율
= 산출세액
  - 세액공제·감면
  - 기납부세액
= 차감납부(환급)세액
```

## 법인세율 (2024년 기준)

| 과세표준 | 세율 |
|---|---|
| 2억 이하 | 9% |
| 2억 초과 ~ 200억 이하 | 19% |
| 200억 초과 ~ 3,000억 이하 | 21% |
| 3,000억 초과 | 24% |

※ 중소기업: 2억 이하 9%, 초과분 19% (사실상 2단계)

## 이월결손금 공제 한도

- **중소기업**: 해당 사업연도 소득의 **100%** (전액 공제 가능)
- **그 외 법인**: 해당 사업연도 소득의 **80%**
- 발생 연도로부터 **15년** 이내 공제 가능 (2020년 이후 발생분)
- 2009~2019년 발생분: 10년 한도

## 주요 익금산입 항목 (세무조정 가산)
- 자산수증이익 (법 제15조)
- 무상으로 받은 자산 / 채무 면제이익
- 임대보증금 간주임대료
- 법인세 환급액 (이미 손금처리 했던 경우)
- 업무무관자산 매각이익

## 주요 손금불산입 항목 (세무조정 가산)
- 법인세·지방소득세
- 과태료·가산세·벌금
- **접대비 한도 초과액**: 한도 = 기본한도(중소 3,600만, 그 외 1,200만) + 수입금액별 추가한도
- **감가상각비 한도 초과액**: 내용연수·상각방법별 한도 초과분
- **업무무관 가지급금 인정이자**: 가지급금 × 당좌대출이자율(현행 4.6%)
- 임원 상여금 한도 초과액 (정관·주주총회 미결의분)
- 주주·임원 퇴직금 한도 초과액

## 감가상각 핵심

| 자산 | 방법 | 내용연수 예시 |
|---|---|---|
| 건물 | 정액법만 | 40년 (철근콘크리트) |
| 기계장치 | 정액·정률 선택 | 5~8년 |
| 차량운반구 | 정액·정률 선택 | 5년 |
| 소프트웨어(개발비) | 정액법만 | 5년 |
| 영업권 | 정액법만 | 5년 |

세무상 상각범위액 = 취득가액 × 상각률
법인이 결산상 계상한 금액 ≤ 상각범위액 → 손금 인정
초과분 → 손금불산입, 유보 (차기 이후 추인)

## 접대비 손금 한도 계산

기본한도:
- 중소기업: 3,600만원
- 중소기업 외: 1,200만원

수입금액 추가한도 (매출액 기준):
- 100억 이하: 수입금액 × 0.3%
- 100억 초과 500억 이하: 3,000만 + (수입금액 - 100억) × 0.2%
- 500억 초과: 1억1,000만 + (수입금액 - 500억) × 0.03%

※ 문화접대비: 위 한도의 20% 추가

## 주요 세무 기한
- 법인세 신고·납부: 사업연도 종료 후 **3개월** (12월 결산법인 → 3월 31일)
- 중간예납: 상반기 종료 후 **2개월** (8월 31일), 직전 연도 산출세액의 50%
- 수정신고: 경정청구 기한 내 (법정신고기한 후 **5년**)
- 경정청구: 법정신고기한 후 **5년** 이내
