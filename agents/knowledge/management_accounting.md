# [KNOWLEDGE] 관리회계·원가관리 핵심 (Planby 전략·재무 분석용)

## 원가 분류

### 행태별 분류
```
고정원가(FC): 조업도 변화에 무관하게 일정 (임차료, 보험료, 감가상각)
변동원가(VC): 조업도에 비례 (재료비, 직접노무비)
준변동원가: 기본 고정 + 변동 혼합 (전기요금 = 기본요금 + 사용량)
준고정원가: 계단 형태 (감독자 인건비)
```

### 관련성별 분류
- **관련원가**: 의사결정에 차이가 나는 미래원가 (증분원가)
- **매몰원가**: 이미 발생, 회수 불가 → 의사결정 무관
- **기회원가**: 포기한 대안의 최대 이익

## CVP 분석 (원가-조업도-이익)

### 공헌이익 구조
```
공헌이익(CM) = 매출액 - 변동원가
공헌이익률(CMR) = CM / 매출액

영업이익 = CM - 고정원가
         = 매출액 × CMR - FC
```

### 손익분기점 (BEP)
```
BEP(수량) = FC / (단가 - 단위변동원가)
          = FC / 단위공헌이익

BEP(금액) = FC / CMR

목표이익 달성 판매량 = (FC + 목표영업이익) / 단위CM
목표이익 달성 매출액 = (FC + 목표영업이익) / CMR
```

### 안전한계
```
안전한계 = 실제(예산)매출액 - BEP 매출액
안전한계율 = 안전한계 / 실제(예산)매출액
영업레버리지도(DOL) = CM / 영업이익 = 1 / 안전한계율
```

## 표준원가 및 차이 분석

### 직접재료비 차이
```
가격차이 = (실제단가 - 표준단가) × 실제투입량
수량차이 = (실제투입량 - 표준투입량) × 표준단가
```

### 직접노무비 차이
```
임률차이 = (실제임률 - 표준임률) × 실제시간
효율차이 = (실제시간 - 표준시간) × 표준임률
```

### 제조간접비 차이 (2분법)
```
예산차이 = 실제 FOH - 변동예산 FOH
조업도차이 = 변동예산 FOH - 표준배부 FOH (고정간접비 기준조업도 차이)
```

## 활동기준원가계산 (ABC)

### 전통 vs ABC
| 항목 | 전통 원가계산 | ABC |
|---|---|---|
| 배부기준 | 조업도(직접노무시간 등) | 활동별 원가동인 |
| 왜곡 | 고정비 왜곡 심함 | 더 정확 |
| 복잡성 | 단순 | 복잡 |
| 적합 상황 | 단일제품·단순공정 | 다품종·서비스업 |

### ABC 4단계
```
1. 활동 식별 (주문처리, 설계변경, 검사 등)
2. 활동별 원가 집계
3. 원가동인 파악 (주문건수, 설계변경횟수 등)
4. 제품·서비스에 원가 배분
```

## 의사결정 분석

### 특별주문 수락 여부
```
기준: 특별주문 단가 > 변동원가 + 기회원가
특수 조건: 여유조업도 있을 때 고정원가 무시
```

### 제품 라인 폐지
```
폐지 유리: 폐지로 절감되는 원가 > 폐지로 상실되는 CM
회피 불가 고정원가 주의 (공통고정원가는 제품 폐지해도 계속 발생)
```

### 내제 vs 외주 (Make or Buy)
```
내제 유리: 내제 변동원가 < 외주단가 - 회피가능 고정원가
외주 시 유휴설비 대안적 이용 가능하면 기회원가 포함
```

## 성과평가 지표

### 투자센터 평가
```
ROI = 영업이익 / 투자자산
RI (잔여이익) = 영업이익 - (투자자산 × 최저요구수익률)
EVA = NOPAT - (WACC × 투자자본)
```

### BSC 재무관점 KPI
| 전략 목표 | KPI | 산식 |
|---|---|---|
| 매출 성장 | 매출성장률 | (당기-전기)/전기 |
| 수익성 개선 | 영업이익률 | 영업이익/매출 |
| 자본효율 | ROIC | NOPAT/투자자본 |
| 현금창출 | FCF | 영업CF - CAPEX |

## 예산 편성 방식

| 방식 | 특징 | 장단점 |
|---|---|---|
| 증분예산 | 전년도 기준 + 증감 | 간단, 기존 비효율 답습 |
| ZBB (영기준예산) | 매년 처음부터 편성 | 효율적, 시간·비용 多 |
| 롤링예산 | 분기마다 1년 앞으로 갱신 | 유연, 관리 부담 |
| 활동기준예산 | ABC 연계 | 정확, 복잡 |

### 예산 vs 실적 차이 분석 흐름
```
매출차이 = 가격차이 + 판매량차이
판매량차이 = 시장점유율차이 + 시장규모차이
원가차이 → 가격차이 + 효율차이 (재료·노무·간접비별)
```
