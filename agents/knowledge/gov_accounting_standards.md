# [KNOWLEDGE] 정부회계기준 핵심 (국가·지방자치단체)

## 정부회계 기준 체계

| 기준 | 적용 대상 | 근거 법령 |
|---|---|---|
| **국가회계기준** | 중앙정부(부처·청) | 국가회계법, 국가회계기준에 관한 규칙 |
| **지방자치단체 회계기준** | 광역·기초 지자체 | 지방자치단체 회계기준에 관한 규칙 |
| **공공기관 회계기준** | 공기업·준정부기관 | K-IFRS 또는 일반기업회계기준 적용 |

---

## 핵심 특징: 발생주의·복식부기

### 도입 배경
- 2009년: 국가 발생주의 도입 (현금주의 → 전환)
- 2007년: 지방자치단체 발생주의 도입

### 현금주의 vs 발생주의 차이

| 항목 | 현금주의 | 발생주의 |
|---|---|---|
| 수익 인식 | 현금 수입 시 | 경제적 사건 발생 시 |
| 비용 인식 | 현금 지출 시 | 자원 소비 시 |
| 감가상각 | 없음 | 있음 |
| 충당부채 | 없음 | 있음 |
| 부채 범위 | 차입금 중심 | 연금부채·보증부채 포함 |

---

## 국가재무제표 구성

| 재무제표 | 내용 |
|---|---|
| **재정상태표** | 자산·부채·순자산 (=민간 재무상태표) |
| **재정운영표** | 수익·비용 (=민간 손익계산서) |
| **순자산변동표** | 순자산 증감 내역 |
| **현금흐름표** | 현금 유출입 |

### 순자산 구성 (민간 자본과 다름)

```
순자산 = 기본순자산 + 적립금 및 잉여금 + 순자산조정
```

- **기본순자산**: 출연금·무상이전 등 원천 자산
- **적립금 및 잉여금**: 누적 재정운영 결과
- **순자산조정**: 투자증권 평가손익 등 기타포괄항목

## 자산 분류 및 인식

### 유형자산

| 항목 | 내용 |
|---|---|
| 인프라 자산 | 도로·철도·항만·댐 (감가상각 대상, 단 비소모성 토지 제외) |
| 사회기반시설 | 국방·보안시설 포함 |
| 일반 유형자산 | 건물·기계장치·차량 등 민간과 동일 |
| 문화재·예술품 | 역사적·문화적 자산 → 상각 면제 |

### 감가상각 (정액법 원칙)

| 자산 | 내용연수 |
|---|---|
| 건물 (콘크리트) | 40년 |
| 건물 (조립식) | 20년 |
| 차량·장비 | 5년 |
| 컴퓨터·전산장비 | 5년 |
| 도로 (포장) | 10~20년 |

### 유산자산
- 문화재, 자연환경보전지역 등 → 재정상태표 표시 (평가액 공시)
- 감가상각 없음

## 부채 분류

### 주요 부채 항목 (민간과 다른 것)

| 항목 | 내용 |
|---|---|
| **국채** | 중앙정부 채무, 만기·이자 명시 |
| **연금충당부채** | 공무원·군인연금 미래 지급 의무 현재가치 → 규모 매우 큼 |
| **보증충당부채** | 정부보증 실행 가능성 × 금액 |
| **퇴직급여충당부채** | 공무원 퇴직금 |

> ⚠️ 연금충당부채: 국가재무제표상 가장 큰 부채 항목. 수백조 규모. 보험수리적 가정(할인율·사망률)에 민감.

## 수익 인식 (재정수입)

### 비교환수익 (정부 특유의 수익)
- **조세수익**: 세금 부과 시점 인식 (현금 수취 시 아님)
- **부담금**: 법적 의무 발생 시
- **기부금**: 수령 시

### 교환수익
- 재화·용역 제공 대가 → 민간과 동일 방식

---

## 예산 vs 재무제표 차이 (핵심 혼동 포인트)

| 구분 | 예산 | 재무제표 |
|---|---|---|
| 기준 | 현금주의 (지출 기준) | 발생주의 |
| 목적 | 자원 배분·통제 | 재무상태 파악 |
| 감가상각 | 없음 | 있음 |
| 연금부채 | 미반영 | 반영 |
| 결산 차이 | 예산집행잔액 | 재정운영결과 |

> 예산 흑자 ≠ 재무제표 흑자. 감가상각·연금부채 때문에 재무제표상 적자인 경우 多.

## 감사원 감사 주요 관점

### 합규성 감사
- 예산 목적 외 사용, 위법·부당 집행 여부
- 법령·규정 준수 여부

### 성과감사
- 사업 목표 달성 여부 (효율성·효과성·경제성)
- 3E: Economy, Efficiency, Effectiveness

### 재무감사
- 국가재무제표 적정성 검토
- 내부회계관리제도 운영 평가

## 지방자치단체 회계 특이사항

### 회계 구분
```
지방자치단체 회계
├─ 일반회계 (일반 행정)
├─ 특별회계 (특정 사업·기금)
│   ├─ 사업별 특별회계
│   └─ 기타 특별회계
└─ 기금 (목적 적립)
```

### 통합재무제표
- 일반·특별회계·기금 통합 → 내부거래 제거 후 작성
- 연결 범위: 출자·출연 기관 포함 여부 검토 필요

### 지방채
- 지방채 발행 한도: 전년도 예산액의 10% 이내 (지방재정법)
- 승인 없이 초과 발행 시 위법
