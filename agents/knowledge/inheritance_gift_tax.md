# [KNOWLEDGE] 상속세 및 증여세 핵심 (상증세법)

## 세율 구조 (상속세·증여세 동일)

| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| 1억 이하 | 10% | — |
| 1억 초과 ~ 5억 이하 | 20% | 1,000만원 |
| 5억 초과 ~ 10억 이하 | 30% | 6,000만원 |
| 10억 초과 ~ 30억 이하 | 40% | 1억6,000만원 |
| 30억 초과 | 50% | 4억6,000만원 |

```
산출세액 = 과세표준 × 세율 - 누진공제
예) 과세표준 20억 → 20억 × 40% - 1억6,000만 = 6억4,000만원
```

## 상속세 공제 항목

### 기본 공제
- **기초공제**: 2억원
- **인적공제**: 자녀 1인당 5,000만, 미성년자 1천만×잔여연수, 연로자(65세↑) 5,000만, 장애인 1천만×잔여기대수명
- **일괄공제**: 기초+인적공제 합계가 5억 미만이면 **5억 선택 가능** (실무상 대부분 일괄공제 적용)

### 배우자상속공제
- 배우자가 실제 상속받은 금액 (한도: 법정상속분 vs 30억 중 적은 금액)
- **최소 5억** 보장 (배우자가 상속받은 재산 없어도 5억 공제)
- 전략: 배우자에게 최대한 상속 → 공제 최대화 가능

### 기타 공제
- **금융재산 상속공제**: 순금융재산 × 20% (한도 2억)
- **동거주택 상속공제**: 10년 이상 동거 + 무주택 요건 충족 → 주택가액의 100% (한도 6억)
- **재해손실공제**: 신고기한 내 재해로 멸실·손상된 재산가액

### 가업상속공제 (핵심 절세 수단)
```
공제 한도:
  - 가업 영위기간 10년 이상: 200억
  - 20년 이상: 400억
  - 30년 이상: 600억

적용 요건:
  - 피상속인: 10년 이상 경영 + 최대주주 지분 50% 이상(상장 30%)
  - 상속인: 상속 후 5년간 가업 종사 + 대표이사 취임
  - 중소·중견기업만 적용 (매출액 5,000억 미만)

사후관리 (5년):
  - 가업 폐업·주식 처분 → 공제액 전액 추징
  - 고용유지 의무: 정규직 80% 이상 유지
```

## 증여세 재산공제 (10년 합산)

| 증여자 | 공제한도 |
|---|---|
| 배우자 | **6억원** |
| 직계존속 (부모·조부모) | **5,000만원** (미성년자 **2,000만원**) |
| 직계비속 (자녀) | **5,000만원** |
| 기타 친족 (형제자매 등) | **1,000만원** |

※ 10년 내 동일인으로부터 받은 증여액 합산 → 공제 중복 불가

### 증여 vs 상속 선택 기준
```
증여 유리: 미래 가치 상승 예상 자산 (주식, 부동산)
           → 낮은 가격에 미리 이전 → 미래 상승분 비과세

상속 유리: 현재 가치 높은 자산 (현금)
           → 배우자공제 최대 활용 가능

할증과세: 수증자가 직계비속인 경우
  - 증여자의 자녀에게 건너뛰어 손자에게 증여 → 30% 할증 (미성년자 40억 초과분 40%)
```

## 비상장주식 증여 (세무조사 위험 높음)

```
평가 방법: 보충적 평가법 (valuation_formulas.md 참조)
주의사항:
  - 저가양도: 시가의 30% 이상 or 3억 이상 차이 → 증여세 과세
  - 고가매입: 시가보다 높게 매수 → 차액 증여
  - 일감몰아주기: 수혜법인 세후영업이익 × 특수관계법인 거래비율 초과분 → 증여세
  - 일감떼어주기: 2014년 이후 신설, 사업기회 제공
```

## 신고·납부 기한

| 구분 | 신고기한 |
|---|---|
| 상속세 | 상속개시일이 속하는 달의 말일부터 **6개월** (비거주자 9개월) |
| 증여세 | 증여받은 날이 속하는 달의 말일부터 **3개월** |

### 연부연납 (분납 제도)
- 산출세액 2,000만 초과 시 신청 가능
- 최대 **10년** 분할납부 (가업상속재산: 20년)
- 연이율: 연 2.9% (2024년 기준) 이자 납부

### 물납
- 부동산·유가증권으로 납부 가능 (현금 납부 곤란 시)
- 납세지 관할 세무서장 허가 필요

## 상속세 절세 핵심 전략 요약

```
1. 사전증여: 10년 단위로 공제 활용, 가치 상승 전 이전
2. 배우자공제 극대화: 배우자 법정상속분 충분히 배정
3. 가업상속공제: 중소·중견기업 → 최대 600억 비과세
4. 동거주택공제: 1세대 1주택 10년 동거 요건 충족
5. 증여재산 합산 주의: 10년 이내 사전증여 → 상속세 합산
   (단, 미리 냈던 증여세는 공제)
```
