# [KNOWLEDGE] 상법 회사편 핵심

## 회사 종류

| 종류 | 특징 | 실무 빈도 |
|---|---|---|
| **주식회사** | 주주 유한책임, 주식으로 자본 구성 | 가장 보편적 |
| 유한회사 | 사원 유한책임, 출자금 (주식 없음), 비공개 운영 | 외국계 자회사, 가족기업 |
| 유한책임회사 | LLC형, 구성원 유한책임, 운영 자율성 높음 | 스타트업, 합작법인 |
| 합명회사 | 사원 무한책임 | 드물음 |
| 합자회사 | 무한책임+유한책임 사원 혼합 | 드물음 |

## 주식회사 핵심 구조

### 자본금
- 최저 자본금 제한 없음 (2009년 폐지)
- 액면가: 100원 이상 (실무: 500원 또는 5,000원)
- **무액면주식**: 발행 가능 (발행가 전부 자본금 또는 일부를 자본잉여금 처리)

### 이사회
- 원칙: 이사 **3인 이상** (자산 2조↑ 상장법인: 사외이사 과반)
- 소규모 회사 (자본금 10억 미만): **1인 또는 2인** 이사 허용
- 이사 임기: 최대 3년 (감사: 3년)
- 대표이사: 이사회에서 선임

### 주주총회 결의 요건

| 구분 | 요건 |
|---|---|
| **보통결의** | 출석 의결권 과반수 + 발행주식 25% 이상 |
| **특별결의** | 출석 의결권 2/3 이상 + 발행주식 1/3 이상 |

특별결의 사항: 정관 변경, 자본 감소, 합병·분할, 영업양도, 이사·감사 해임

## 종류주식 (스타트업 투자 핵심)

```
1. 우선주 (Preferred Stock)
   - 배당·잔여재산 분배 우선권
   - 상환우선주(RCPS): 일정 조건 시 회사가 상환 가능
   - 전환우선주(CPS): 보통주 전환 가능

2. 전환주식: 다른 종류 주식으로 전환 가능

3. 상환주식: 주주 청구 or 회사 일방으로 상환 가능
   RCPS = 상환전환우선주 (가장 흔한 VC 투자 방식)

4. 의결권 없는 주식 (무의결권 우선주)
   - 한도: 발행주식 총수의 1/4

5. 복수의결권주식 (차등의결권, 2023년 도입)
   - 비상장 벤처기업 한정
   - 주당 최대 10배 의결권
   - 창업자 지분 희석 방지 목적
```

## 주식 발행 및 자금조달

### 신주 발행 (유상증자)
```
원칙: 기존 주주에게 지분율 비례 배정 (주주배정)

제3자배정 (투자 유치):
  - 정관에 근거 있거나 주주총회 특별결의
  - 스타트업 투자는 대부분 제3자배정 방식

발행가격:
  - 상장사: 시가 기준 (이사회 결의)
  - 비상장사: 보충적 평가액 (상증세법) or 협의
```

### 주주간계약 (SHA, Shareholders' Agreement)
```
주요 조항:
  1. 경영권 보호:
     - 대표이사 지명권
     - 이사회 구성 (투자자 이사 파견)
     - 거부권 (Veto Right): 중요 의사결정 동의 요건

  2. 주식 이전 제한:
     - 우선매수권 (ROFR): 매도 전 기존 주주에게 우선 제안 의무
     - 동반매도권 (Tag-Along): 창업자 매각 시 같은 조건으로 따라 팔 권리
     - 동반매도요구권 (Drag-Along): 인수자가 100% 요구 시 강제 매각
     - 락업 (Lock-up): 일정 기간 매각 금지

  3. 투자자 보호:
     - 반희석화 조항 (Anti-dilution): 후속 투자 시 지분율 보호
       - Full Ratchet: 낮은 가격으로 전량 재조정 (강한 보호)
       - Weighted Average: 가중평균으로 조정 (표준)
     - 청산우선권 (Liquidation Preference): M&A·청산 시 우선 회수
     - 참여권 (Participation Right): 청산우선권 회수 후 잔여분 추가 참여
```

## 합병·분할

### 합병
```
종류:
  - 흡수합병: A + B → A (B 소멸)
  - 신설합병: A + B → C (A, B 모두 소멸)

절차:
  1. 이사회 결의 → 합병계약 체결
  2. 주주총회 특별결의 (양사)
  3. 채권자 이의절차 (1개월 이상 공고)
  4. 합병 등기
  5. 소규모합병: 순자산가액 5% 이하 → 주총 생략 가능
  6. 간이합병: 모회사가 자회사 90% 이상 보유 → 자회사 주총 생략

합병비율: 각사 주식가치 기준으로 산정 (비상장 시 보충적 평가법)
```

### 분할
```
인적분할 (spin-off):
  - 사업부문을 별도 법인으로 분리
  - 기존 주주가 신설법인 주식 직접 취득
  - 세금: 요건 충족 시 과세이연 가능 (물적분할과 차이)

물적분할:
  - 사업부문을 자회사로 분리
  - 모회사가 자회사 주식 100% 취득
  - 자회사 IPO 시 활용 (예: LG에너지솔루션)
```

## 자기주식 (자사주)

```
취득 가능: 배당가능이익 범위 내
방법: 거래소 매입, 공개매수, 특정주주 매입

활용:
  - 주가 방어 (EPS 증가)
  - 임직원 주식매수선택권(스톡옵션) 지급용
  - M&A 시 교환 대가

소각: 이사회 결의로 소각 → 발행주식 수 감소 → 주당가치 상승
주의: 자기주식 의결권 없음
```

## 영업양도 vs 합병 비교

| 구분 | 영업양도 | 합병 |
|---|---|---|
| 이전 방식 | 계약으로 개별 자산·부채 이전 | 포괄승계 (자동) |
| 부채 | 선택적 인수 가능 | 전부 승계 |
| 세금 | 법인세·부가세 과세 | 요건 충족 시 과세이연 |
| 절차 | 상대적 간단 | 주총·채권자 절차 복잡 |
| 숨은 부채 리스크 | 낮음 (선택 인수) | 높음 (전부 승계) |

## 스타트업 실무 빈출 이슈

### 주식매수선택권 (스톡옵션)
```
부여 한도: 발행주식 총수의 10% (벤처기업 50%)
행사가격: 부여일 시가 이상 (비상장: 보충적 평가액 이상)
행사기간: 부여 후 2년 이상 재직 후 행사 가능

세금:
  - 행사 시: 시가 - 행사가 = 근로소득세 과세 (원천징수)
  - 벤처기업 인정 옵션: 연 5천만 이하 행사분 비과세
```

### 전환사채(CB)·신주인수권부사채(BW)
```
CB: 일정 조건 충족 시 주식 전환
  - 전환가격: 보통 발행 시 주가 (또는 협의)
  - 리픽싱: 주가 하락 시 전환가격 하향 조정 (투자자 보호)
  - 발행: 이사회 결의 (주총 위임 가능)

BW: 신주인수권만 분리 행사 가능 (채권은 유지)
  - 비상장사 발행 多 → 신주인수권 거래 가능
```

### 주주명부 vs 실질 주주
```
원칙: 주주명부에 기재된 자가 주주권 행사
명의신탁 위험: 차명주식 → 증여세 과세 + 과태료
정관: 주식 양도 제한 조항 → M&A 전 정관 확인 필수
```
