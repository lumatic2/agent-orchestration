# [KNOWLEDGE] 회계감사 핵심 기준 및 실무

## 감사위험 모형

```
감사위험(AR) = 고유위험(IR) × 통제위험(CR) × 발견위험(DR)

목표: AR ≤ 허용 감사위험 수준 (통상 5%)

DR = AR / (IR × CR)
→ IR·CR 높을수록 DR 낮춰야 함 → 실증절차 강화
```

## 중요성 기준 (Materiality)

전체 중요성(PM) 산정 기준값:
- 세전이익의 5~10%
- 매출액의 0.5~1%
- 자산총계의 0.5~1%
- 자본총계의 1~2%

수행 중요성(TE): PM의 **50~75%** (발견 누락 위험 완충)
명세 중요성: 특정 항목(현금·관계자 거래)은 별도 낮은 기준 적용

## 핵심감사사항 (KAM)

기재 요건 (회계감사기준 제701호):
- 감사인 판단상 당기 재무제표 감사에서 가장 유의적인 사항
- 의사소통 필요 사항 중 선별
- 상장법인 감사보고서에 필수 기재 (2020년 시행)

KAM 기재 내용:
1. 핵심감사사항으로 결정한 이유
2. 어떻게 감사에서 다루었는지
3. 관련 재무제표 공시사항 참조

주요 KAM 유형: 수익 인식, 영업권 손상, 금융상품 공정가치, 충당부채 추정

## 내부회계관리제도 (ICFR)

### 적용 기준 (자산 규모별)

| 자산 규모 | 감사인 의무 |
|---|---|
| 2조원 이상 (상장) | **감사** (인증 수준 최고) |
| 5천억 이상 (상장) | **감사** |
| 1천억 이상 (상장) | **감사** |
| 1천억 미만 (상장) | **검토** |
| 비상장 외감대상 | 경영진 평가 (감사인 검토 없음) |

### COSO 내부통제 5요소
1. 통제환경 (Control Environment)
2. 위험평가 (Risk Assessment)
3. 통제활동 (Control Activities)
4. 정보·의사소통 (Information & Communication)
5. 모니터링 활동 (Monitoring)

### 미비점 분류

| 구분 | 정의 | 영향 |
|---|---|---|
| 단순미비점 | 합리적으로 가능한 허위표시 방지·발견 실패 가능성 있음 | 공시·감사의견 영향 없음 |
| 유의미비점 | 경영진 주의가 필요할 정도의 단순미비점 | 지배기구에 서면 통보 |
| **중요취약점** | 중요한 허위표시 방지·발견 실패 가능성이 합리적으로 존재 | **비적정 감사의견** 가능 |

## 감사의견 유형

| 의견 | 조건 |
|---|---|
| **적정** | 중요한 왜곡표시 없음 |
| **한정** | 특정 사항에 국한된 중요한 왜곡표시 |
| **부적정** | 전반적으로 중요하고 광범위한 왜곡표시 |
| **의견거절** | 감사범위 중요한 제한 OR 계속기업 불확실성 심각 |

## 표준감사시간 제도 (2019년 도입)

- 한공회가 업종·규모별 표준시간 고시
- 감사인은 표준시간의 **일정 비율 이상** 감사시간 투입 의무
- 미달 시 품질관리감리 대상

## 계속기업 가정 평가 (회계감사기준 제570호)

의심 징후:
- 순손실 지속, 현금부족, 차입금 만기 도래
- 주요 고객 상실, 핵심 인력 이탈
- 법적 분쟁 (손해배상 위험)

감사인 절차:
1. 경영진의 평가 검토
2. 경영진 계획의 실행 가능성 검토
3. 추가 정보 수집 (이후 기간 차입약정 등)
4. 감사보고서 효과 결정 (중요한 불확실성 → 강조사항 기재)

## 외감법 주요 규정 (주식회사 등의 외부감사에 관한 법률)

- 외감 대상: 직전 연도 자산 120억 이상 or 매출 100억 이상 등
- 주기적 지정제: 상장법인 6년 자유선임 + 3년 지정
- 감사인 등록제: 상장법인 감사는 등록 감사인만 가능
- 감사보고서 제출: 사업보고서 제출 기한 (상장 90일, 비상장 120일)
- 감리: 금융감독원이 감사보고서 감리, 재무제표 심사
