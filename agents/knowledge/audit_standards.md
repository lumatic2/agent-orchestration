## 감사위험 모형 및 중요성

### 감사위험 모형
```
AR = IR × CR × DR

목표 AR ≤ 5% (통상)
DR = AR / (IR × CR)
→ IR·CR 높을수록 DR ↓ → 실증절차 강화
```

| IR·CR 조합 | DR 목표 | 대응 강도 |
|---|---|---|
| 높음 × 높음 | 매우 낮음 (<15%) | 광범위한 실증절차 |
| 높음 × 낮음 | 낮음 (15~35%) | 실증절차 확대 |
| 낮음 × 높음 | 낮음 | 실증절차 확대 |
| 낮음 × 낮음 | 중간 (35~60%) | 표준 실증절차 |

### 중요성 설정 기준
| 기준값 | 비율 범위 | 사용 상황 |
|---|---|---|
| 세전이익 | 5~10% | 이익 실현 기업 (가장 일반적) |
| 매출액 | 0.5~1% | 이익 변동 크거나 손실 기업 |
| 자산총계 | 0.5~1% | 금융업·지주회사 |
| 자본총계 | 1~2% | 자본 중심 평가 기업 |
| 총비용 | 1~2% | 비영리·공공기관 |

- **수행 중요성(TE)**: PM의 50~75% 적용
- **명세 중요성**: 현금·관계자거래·임원보수 등 낮은 별도 기준

## 샘플링 방법론

### 속성 샘플링 (Attribute Sampling) — ICFR 운영평가
| 신뢰수준 | 허용오류율 5% | 허용오류율 10% |
|---|---|---|
| 90% | 45개 | 22개 |
| 95% | 59개 | 29개 |
| 99% | 90개 | 44개 |

- **예상오류율 0% 전제** — 오류 발견 시 표본 추가
- **빈도별 최소 표본**: 연 1회 통제 = 1개, 월 = 2개, 주 = 5개, 일 = 25개

### 통화단위 샘플링 (MUS — Monetary Unit Sampling)
```
표본 크기 = (모집단 금액 × 신뢰계수) / (PM × 예상왜곡표시비율)

신뢰계수 (95% 신뢰수준):
- 허용왜곡표시비율 = PM 기준
- 예상왜곡표시 없을 때: 3.0
- 예상왜곡표시 5% 시: 약 4.0
```

**MUS 장점**: 고액 항목 자동 포함, 금액 비례 선정
**MUS 단점**: 과소계상(understatement) 탐지 취약

### 판단 샘플링 기준
- 고위험 거래처·임원·관계자 → 전수 확인
- 이상 항목 (기말 대형 거래, 역분개, 일회성 항목) → 전수 확인
- 나머지: 금액 상위 or 무작위

## Walkthrough 절차

목적: 내부통제의 설계 적정성 평가 + 업무이해
대상: 각 핵심 프로세스(수익, 매입, 급여, IT 등) 1~2건

```
절차 흐름:
1. 거래 원시 발생 (주문서·계약서)
   ↓
2. 시스템 입력 (ERP 화면 직접 확인)
   ↓
3. 승인 절차 (결재선·전자서명 확인)
   ↓
4. 재무제표 반영 (계정·금액 추적)
   ↓
5. 통제 수행 증거 확인 (승인 도장·시스템 로그)
```

**문서화**: 프로세스 흐름도 + 각 단계 스크린샷·증빙 첨부

## IT 일반통제 (ITGC) 감사

### 4대 영역 세부 테스트
| 영역 | 주요 통제 | 테스트 방법 |
|---|---|---|
| **접근통제** | 계정 생성·삭제·권한 | 활성 계정 목록 vs 재직자 대조, 퇴사자 계정 확인 |
| **변경관리** | 프로그램 변경 승인·테스트·적용 | 변경이력 로그, 승인문서 샘플 |
| **운영통제** | 배치잡 모니터링, 오류 처리 | 배치 로그 확인, 오류 재처리 기록 |
| **백업·복구** | 백업 주기·검증, DR테스트 | 백업 로그, DR테스트 결과 문서 |

**IT 의존도 높을수록** ITGC 미비 → 전산 산출물 신뢰 불가 → 수작업 대체 필요

### IPE (Information Produced by Entity)
- 회사가 자체 생산한 보고서·데이터 활용 시 IPE 정확성·완전성 검증 필수
- 검증: 원본 시스템 쿼리 결과 vs 보고서 합계 대조

## 계정과목별 실증절차

### 매출·매출채권
| 절차 | 목적 | 방법 |
|---|---|---|
| 외부조회 (KSA 505) | 실재성·완전성 | 주요 고객에 잔액 확인서 발송 |
| 후속 입금 확인 | 실재성 | 기말 후 실제 수금 내역 추적 |
| 수익인식 기준 검토 | 기간귀속 | 계약서·납품서·인수확인서 대조 |
| 절사 테스트 (Cut-off) | 기간귀속 | 기말 전후 2주 거래 상세 검토 |
| 매출채권 연령분석 | 평가 | 연체 비율, 충당금 적정성 |

### 재고자산
| 절차 | 목적 |
|---|---|
| 실사 참관 | 실재성·완전성 |
| 단가 검증 | 평가 (원가 vs 순실현가능가치) |
| 재고 조정 내역 검토 | 완전성 |
| 진부화 평가 | 평가 |

### 고정자산 (유형·무형)
- 자본적 지출 vs 수익적 지출 분류 검토
- 감가상각률·내용연수 합리성 검토
- 손상 징후 식별 및 회수가능액 검토
- 개발비 자본화 요건 충족 여부 (IAS 38 6가지 기준)

### 충당부채 (KSA 540)
- 추정 방법론 적정성 검토
- 전기 추정 대비 실제 결과 비교 (PMR: Post-Model Review)
- 경영진 편의(Bias) 평가: 지속적 과소·과대 추정 패턴
- 외부 전문가(계리사·변호사) 의견 적정성

## 금감원 감리 주요 반복 지적 사례

### 수익 인식
- 반품 가능 매출 조기 인식
- 대리인 거래를 본인으로 총액 표시
- 진행기준 투입법 사용 시 투입비용 과다 계상
- 기말 집중 매출 (channel stuffing)

### 자산 과대 계상
- 개발비 자본화 요건 미충족 (IAS 38 — 기술적 실현 가능성 미입증)
- 손상 미인식: 영업권, 무형자산, 투자주식
- 재고 순실현가능가치 미적용

### 비용 조작
- 수익적 지출 자본화 (유형자산·소프트웨어)
- 충당부채 과소 설정 (미지급비용 누락)
- 관계사 간 비용 전가 (고가 용역 매입)

### 공시 누락
- 우발부채 미공시 (소송, 보증)
- 특수관계자 거래 불완전 공시
- 계속기업 불확실성 미기재

### 최근 테마감리 이슈 (2022~2024)
- 가상자산 회계처리 (자산 분류·공정가치)
- ESG 공시 관련 회계 영향 (탄소배출권)
- IFRS 17 보험계약 초기 적용 오류

## 감사의견 결정 기준

| 상황 | 의견 유형 |
|---|---|
| 중요한 왜곡표시 없음 | 적정 |
| 특정 사항 왜곡표시, 범위 한정적 | 한정 |
| 왜곡표시 중요하고 광범위 | 부적정 |
| 감사범위 제한 심각 (중요하고 광범위) | 의견거절 |
| 계속기업 불확실성 심각 | 의견거절 (또는 강조사항 + 적정) |
| ICFR 중요취약점 존재 | 비적정 내부회계 의견 |

**한정 vs 부적정 기준**: 왜곡표시가 재무제표 "전반"에 영향을 미치는지 여부

## 산업별 핵심 감사 포인트

### 건설업
- 진행률 산정 방식 (투입법·산출법) 적정성
- 공사손실충당부채: 미래 예상 손실 충분 적립 여부
- 분양계약 수익인식 시점 (IFRS 15 이행의무 충족 시점)
- 공사 관계사 간 내부거래 제거

### 금융업 (은행·보험·증권)
- ECL(기대신용손실): Stage 1/2/3 분류, PD·LGD·EAD 가정 합리성
- 금융상품 공정가치: Level 1→2→3 이동, Level 3 모델 검증
- IFRS 17 (보험): CSM 산정, 적용범위, 이행현금흐름
- 자본비율: BIS·지급여력비율 산정 적정성

### IT/SaaS
- 소프트웨어 자본화: IAS 38 개발단계 판단 (연구단계 비용처리)
- 구독 수익 SSP(독립판매가격) 배분 방식
- 스톡옵션 공정가치 (Black-Scholes 가정: 변동성·무위험이자율)
- ARR/MRR vs GAAP 매출 차이 조정

### 바이오·제약
- 임상단계별 개발비 자본화 가부: 1상(연구단계)·2상(판단)·3상(자본화 가능)
- 기술이전 수익인식: 마일스톤 조건부 대가 처리 (변동대가 추정)
- 원가 구조: 위탁생산(CMO) 계약 원가 귀속

## 계속기업 평가 체크리스트

**재무적 징후:**
- [ ] 2기 연속 순손실
- [ ] 영업현금흐름 마이너스 (3기 이상)
- [ ] 순부채 > 자산총계 (자본잠식)
- [ ] 단기 차입금 만기 (12개월 내) > 보유 현금
- [ ] 주요 차입약정 covenant 위반

**운영적 징후:**
- [ ] 주요 고객(매출 20%↑) 계약 해지
- [ ] 핵심 임원·인력 대규모 이탈
- [ ] 주요 라이선스·허가 취소 위험

**감사보고서 효과:**
- 중요한 불확실성 존재 + 주석 공시 충분 → 강조사항 + 적정의견
- 주석 공시 불충분 → 한정 또는 부적정
- 불확실성 매우 심각, 경영진 계획 신뢰 불가 → 의견거절
