# [KNOWLEDGE] 자본시장·외부감사 핵심

## 자본시장과 금융투자업에 관한 법률 (자본시장법)

### 금융투자상품 분류
```
금융투자상품
├─ 증권 (원본 초과손실 없음)
│   ├─ 지분증권 (주식, 출자증권, 수익증권)
│   ├─ 채무증권 (국채, 회사채, CP)
│   ├─ 수익증권 (펀드)
│   ├─ 투자계약증권
│   └─ 파생결합증권 (ELS, DLS)
└─ 파생상품 (원본 초과손실 가능)
    ├─ 장내파생상품 (선물, 옵션)
    └─ 장외파생상품 (Swap, Forward)
```

### 공시 의무 (상장법인)
| 공시 유형 | 기한 | 주요 내용 |
|---|---|---|
| 사업보고서 | 사업연도 종료 후 90일 | 연간 재무정보 |
| 반기보고서 | 반기 종료 후 45일 | 상반기 재무정보 |
| 분기보고서 | 분기 종료 후 45일 | 분기 재무정보 |
| 주요사항보고서 | 사유 발생 즉시 | M&A, 유상증자 등 |

### 내부자거래 금지
- 미공개 중요정보 이용 금지 (자본시장법 제174조)
- 중요정보: 주가에 중대한 영향 미칠 수 있는 정보
- 위반 시: 1년 이상 징역 또는 이익의 3~5배 벌금

### 시장질서 교란행위 (2015년 신설)
- 부정거래, 시세조종 외에 "질서교란행위"도 규제
- 과징금: 위반 이익의 1.5배 또는 5억원 이하

## 주식회사 등의 외부감사에 관한 법률 (외감법)

### 외부감사 대상
| 구분 | 기준 |
|---|---|
| 주권상장법인 | 전부 의무 |
| 비상장 주식회사 | 자산 120억 이상 OR 부채 70억 이상 OR 매출 100억 이상 OR 종업원 100명 이상 |
| 유한회사 | 자산 500억 이상 등 일정 기준 |

### 감사인 선임
- 상장법인·소유경영 미분리 대형법인: **감사인 주기적 지정제** (6년 자유선임 + 3년 지정)
- 감리: 금융감독원 감리, 품질관리감리

### 내부회계관리제도
- 자산 2조 이상 상장사: 외부감사인의 **인증감사** 의무 (2019년~)
- 자산 5,000억~2조: 검토 → 단계적 확대
- 경영진 평가·이사회 승인·감사위원회 검토 필수

## 주요 금융 규제 지표

### BIS 자기자본비율 (은행)
```
BIS비율 = 자기자본 / 위험가중자산 ≥ 8% (Basel III: 10.5%)
CET1(보통주자본비율) ≥ 4.5%
```

### 유동성커버리지비율 (LCR)
```
LCR = 고유동성자산 / 30일 순현금유출액 ≥ 100%
```

## IPO (기업공개) 절차 핵심

### 코스피 상장 요건 (주요)
| 항목 | 기준 |
|---|---|
| 영업이익 | 최근 3년 합계 50억 이상 |
| 자기자본 | 300억 이상 |
| 주식분산 | 소액주주 500인 이상, 25% 이상 보유 |

### 코스닥 상장 요건
| 항목 | 이익실현기준 | 시가총액기준 |
|---|---|---|
| 기준 | 당기순이익 20억 이상 | 시가총액 500억 이상 |

### IPO 주요 비용
- 주관사 수수료: 공모금액의 1~3%
- 감사·법무·IR 비용: 3~10억 수준
- Lock-up: 최대주주 6개월 보호예수

## 회사채 발행 실무

### 신용등급 체계 (한국신용평가)
```
투자적격: AAA ~ BBB-
투기등급:  BB+ ~ D
```

| 등급 | 의미 |
|---|---|
| AAA | 최우량, 채무이행 확실성 최고 |
| AA | 우량, 원리금 지급 능력 충분 |
| A  | 양호, 경기변동 다소 영향 |
| BBB | 투자적격 최하, 채무이행 능력 적정 |
| BB↓ | 투기등급, 불확실성 높음 |

### 회사채 발행 방식
- **공모**: 금감원 증권신고서 제출, 50인 이상 청약
- **사모**: 49인 이하, 신고 불필요, 기관투자자 중심
- **ABS (자산유동화증권)**: 특수목적법인(SPC) 통해 자산 유동화
