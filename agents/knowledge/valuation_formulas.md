# [KNOWLEDGE] 기업가치평가 핵심 공식

## 상증세법 비상장주식 평가 (보충적 평가법)

### 1주당 평가액 계산

```
1주당 평가액 = (순손익가치 × 3 + 순자산가치 × 2) / 5
```

단, 부동산 과다 법인 등은 순자산가치를 60% 이상 반영

### 순손익가치 계산

```
순손익가치 = 1주당 최근 3년 가중평균 순손익 / 자본환원율(10%)
```

가중평균 순손익:
- 평가기준일 1년 전 결산: 가중치 3
- 평가기준일 2년 전 결산: 가중치 2
- 평가기준일 3년 전 결산: 가중치 1

```
가중평균 순손익 = (1년전 × 3 + 2년전 × 2 + 3년전 × 1) / 6
```

### 순자산가치 계산

```
순자산가치 = (자산총계 - 부채총계) / 발행주식 총수
```
- 자산·부채는 **공정가치(시가)** 기준 (장부가 아님)
- 영업권은 별도 계산: 초과수익 × 3년 환원 (지급이자율 적용)

### 지배주주 할증

- 최대주주 등이 보유한 주식: 평가액의 **20% 할증** (일반 법인)
- 중소기업 최대주주: **10% 할증**
- 코스닥 상장법인 최대주주: **20% 할증**
- 소수주주 지분: 할증 없음

### 계산 예시

```
가정:
- 발행주식: 100,000주
- 순자산(공정가치): 20억원 → 순자산가치 = 20,000원/주
- 가중평균 순손익: 2억원/년 → 1주당 순손익 = 2,000원
- 순손익가치 = 2,000 / 0.1 = 20,000원/주

1주당 평가액 = (20,000 × 3 + 20,000 × 2) / 5 = 20,000원
최대주주 할증(20%) = 20,000 × 1.2 = 24,000원/주
```

## DCF (현금흐름할인법)

### FCFF (세후 잉여현금흐름)

```
FCFF = EBIT × (1 - 세율) + D&A - CapEx - ΔWorking Capital
     = NOPLAT + D&A - CapEx - ΔWorking Capital
```

### WACC 계산

```
WACC = Ke × (E/V) + Kd × (1-t) × (D/V)

Ke = Rf + β × ERP + 규모프리미엄
   = 무위험이자율 + 레버드베타 × 주식위험프리미엄

Kd = 차입금 이자율 (시장이자율 기준)
```

한국 기준값 (2024년 기준 참고):
- 무위험이자율(Rf): 국고채 10년물 (약 3.0~3.5%)
- ERP: 5~6% (국내 시장 프리미엄)
- 레버드베타: 유사 상장사 β 조정

### 터미널 밸류

고든성장모형:
```
TV = FCFn × (1 + g) / (WACC - g)
g: 명목 GDP 성장률 하한 (통상 2~3%)
```

EV/EBITDA Exit Multiple:
```
TV = EBITDAn × 업종 Exit Multiple
```

### 기업가치 → 주주가치

```
EV(기업가치) = PV(FCFF) + TV의 현재가치
주주가치(Equity Value) = EV - 순차입금(Net Debt)
주당가치 = 주주가치 / 발행주식 수
```

## 상대가치평가 (PER 기준, IPO 공모가 산정)

### 유사회사 비교법 (코스닥 기술특례 포함)

```
주당평가액 = EPS(예측) × 유사회사 평균 PER × (1 - 할인율)
할인율: 15~30% (공모가 밴드 하단 기준)
```

### 주요 배수 기준값 (업종별 참고)

| 업종 | EV/EBITDA | PER |
|---|---|---|
| IT/플랫폼 | 15~25x | 25~40x |
| 제조(일반) | 6~10x | 12~18x |
| 바이오(흑자) | 20~30x | 30~50x |
| 금융 | - | 8~12x PBR |
| 건설 | 5~8x | 8~12x |

## PPA (매수가격배분) 핵심 흐름

```
인수가격(취득원가)
- 피취득자 순자산 공정가치
= 영업권(Goodwill) [양수] 또는 염가매수차익 [음수]
```

무형자산 평가 방법:
- **고객관계**: 초과수익법(MEEM) — 고객 유지율·수익성 기반
- **기술/IP**: 로열티절감법 — 시장 로열티율 × 매출 현재가치
- **브랜드**: 로열티절감법 또는 잉여이익법
- **경쟁금지약정**: 차이법 (있을 때 vs 없을 때 현금흐름 차이)

## 전환사채(CB) 가치 분리

```
CB 공정가치 = 부채요소(일반사채 가치) + 자본요소(전환권 옵션 가치)

부채요소 = 동일 신용등급 일반사채 이자율로 할인한 현재가치
자본요소 = CB 발행가 - 부채요소 (잔여법)
```

옵션 가치 산정: Black-Scholes 또는 이항모형
- 주요 파라미터: 주가, 행사가격, 변동성(σ), 무위험이자율, 만기
