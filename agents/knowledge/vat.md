# [KNOWLEDGE] 부가가치세 핵심 구조

## 기본 계산 구조

```
납부세액 = 매출세액 - 매입세액

매출세액 = 공급가액 × 10%
매입세액 = 매입액 × 10% (세금계산서 수취분)
```

### 세율
- **표준세율**: 10%
- **영세율**: 0% (수출, 국제 용역, 외화 획득 재화·용역)
- **면세**: 세금 없음 (매입세액 공제도 안 됨)

---

## 일반과세자 vs 간이과세자

| 구분 | 일반과세자 | 간이과세자 |
|---|---|---|
| 기준 | 연 매출 **1억400만원 이상** | 1억400만원 미만 |
| 납부세액 | 매출세액 - 매입세액 | 공급대가 × 업종별 부가가치율 × 10% |
| 세금계산서 | 발행 의무 | 발행 불가 (영수증만) |
| 환급 | 가능 | 불가 |
| 신고 | 연 2회 (예정신고 포함 4회) | 연 1회 (1월) |

### 간이과세자 업종별 부가가치율
| 업종 | 부가가치율 |
|---|---|
| 소매업, 음식점업 | 15% |
| 제조업, 농·임·어업 | 20% |
| 건설업, 운수·창고업 | 30% |
| 서비스업 (일반) | 40% |
| 금융·보험업 | 50% |

```
예) 소매업 간이과세자, 연 공급대가 8,000만:
납부세액 = 8,000만 × 15% × 10% = 120만원
```

---

## 신고·납부 기한

| 구분 | 과세기간 | 예정신고 | 확정신고 |
|---|---|---|---|
| 1기 | 1월 1일 ~ 6월 30일 | 4월 25일 | **7월 25일** |
| 2기 | 7월 1일 ~ 12월 31일 | 10월 25일 | **다음해 1월 25일** |

- **예정신고**: 직전 과세기간 납부세액의 50% 고지 납부 (또는 자진신고)
- 법인사업자: 예정신고 의무 / 개인 일반과세자: 고지 납부

---

## 영세율 vs 면세 비교

| 구분 | 영세율 | 면세 |
|---|---|---|
| 세율 | 0% | 해당 없음 |
| 매입세액공제 | **가능** (환급 받음) | **불가** |
| 세금계산서 | 발행 의무 | 계산서 발행 |
| 해당 항목 | 수출, 국제운송, 외화용역 | 기초식품, 의료, 교육, 금융, 토지 |

**면세 사업자의 딜레마**: 매입세액 공제 불가 → 원가에 흡수 → 경쟁력 약화

---

## 매입세액 불공제 항목 (중요)

```
1. 비영업용 소형승용차 (배기량 1,000cc 초과 승용차)
   - 구입·임차·유지 관련 매입세액 전액 불공제
   - 예외: 운수업, 자동차 판매업, 운전학원

2. 접대비 관련 매입세액 전액 불공제

3. 면세사업·토지 관련 매입세액

4. 사업과 직접 관련 없는 지출

5. 세금계산서 미수취 or 불분명 세금계산서
   (공급자 사업자번호, 작성일자, 금액 등 필수 기재사항 미기재)
```

---

## 세금계산서 핵심

### 발행 의무
- 사업자가 재화·용역을 공급할 때 **반드시** 발행
- 공급시기 (재화: 인도일, 용역: 완료일)에 발행 원칙
- 전자세금계산서: 법인 및 개인(직전연도 공급가액 1억↑) 의무 발행

### 세금계산서 필수 기재사항
```
1. 공급자·공급받는자 사업자등록번호·성명
2. 공급가액 및 세액
3. 작성연월일
4. 공급하는 사업장 소재지
```

### 수정세금계산서 발행 사유
- 착오·정정, 환입, 계약 해제, 내국신용장 사후 개설

## 부동산 관련 부가세 (실무 빈출)

```
건물 공급: 과세 (건물분만, 토지는 면세)
  → 분양가 중 토지/건물 안분 필요

임대:
  - 주택임대: 면세
  - 상가임대: 과세 (간이과세자 예외 있음)
  - 오피스텔: 주거용=면세, 업무용=과세 (실질 판단)

임대보증금 간주임대료:
  - 보증금 × 기준이율(연 3.5%, 2024) = 간주임대료 → 부가세 과세
```

---

## 수출 영세율 적용 절차

```
1. 수출신고필증 보관 (증빙)
2. 영세율 세금계산서 발행 (공급가액만, 세액 0)
3. 매입세액은 환급 신청
4. 조기환급: 수출 비중 높은 사업자 → 예정신고 기간 중 신청 가능
```

## 사업자등록 및 폐업

| 구분 | 기한 |
|---|---|
| 사업자등록 | 사업 개시일부터 **20일** 이내 |
| 폐업 신고 | 폐업일로부터 **25일** 이내 |
| 폐업 시 재고자산 | 시가로 자가공급 간주 → 부가세 납부 |
