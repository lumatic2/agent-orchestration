# [KNOWLEDGE] 소득세 핵심 (개인·사업소득)

## 소득세 종합과세 구조

### 종합소득 범위
```
종합소득 = 이자소득 + 배당소득 + 사업소득 + 근로소득
         + 연금소득 + 기타소득
```

- 분리과세 소득은 종합소득에 미합산 (원천징수로 납세 종결)
- 양도소득·퇴직소득은 별도 과세

### 소득세율 (2024년 기준)
| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| 1,400만 이하 | 6% | — |
| 1,400만~5,000만 | 15% | 126만 |
| 5,000만~8,800만 | 24% | 522만 |
| 8,800만~1.5억 | 35% | 1,490만 |
| 1.5억~3억 | 38% | 1,940만 |
| 3억~5억 | 40% | 2,540만 |
| 5억~10억 | 42% | 3,540만 |
| 10억 초과 | 45% | 6,540만 |

지방소득세 = 소득세 × 10% (별도 납부)

## 근로소득 핵심

### 근로소득공제
| 총급여액 | 공제액 |
|---|---|
| 500만 이하 | 전액 |
| 500만~1,500만 | 500만 + 초과분×50% |
| 1,500만~4,500만 | 1,000만 + 초과분×15% |
| 4,500만~1억 | 1,450만 + 초과분×5% |
| 1억 초과 | 2,000만 (한도) |

### 주요 세액공제 (근로소득자)
- 근로소득세액공제: 산출세액 × 55~74% (한도 50~74만)
- 자녀세액공제: 1명 25만, 2명 55만, 3명이상 1명당 30만 추가
- 의료비공제: 총급여 3% 초과분 × 15% (난임치료 30%)
- 교육비공제: 자녀 1인당 300만(대학) × 15%
- 기부금공제: 법정·지정기부금 15~30%
- 연금저축+IRP: 최대 900만원 × 12~15%

## 사업소득 핵심

### 장부 작성 의무
| 구분 | 기준 | 방법 |
|---|---|---|
| 복식부기 의무자 | 업종별 수입금액 기준 이상 | 복식부기 |
| 간편장부 대상자 | 신규·소규모 사업자 | 간편장부 |
| 추계 | 장부 미비 | 기준경비율·단순경비율 |

### 기준경비율 vs 단순경비율
```
추계 소득금액:
  기준경비율 적용자: 수입금액 - 주요경비(매입비용+임차료+인건비) - (수입금액×기준경비율)
  단순경비율 적용자: 수입금액 × (1 - 단순경비율)
```

## 금융소득 과세

### 이자·배당소득 과세 방식
- 연간 합계 **2,000만원 이하**: 원천징수(14%)로 분리과세 종결
- **2,000만원 초과**: 전액 종합과세 (초과분만 아님, 전액 합산)
- 금융소득종합과세 시 비교과세: Max(종합과세, 분리과세세액)

### 배당소득 Gross-up
- 법인세 이중과세 조정: 배당 × 11% = 귀속법인세
- 총수입금액 = 배당 + 귀속법인세
- 배당세액공제 = 귀속법인세 (산출세액 범위 내)

## 양도소득세 핵심

### 부동산 양도소득세율
| 보유기간 | 세율 |
|---|---|
| 1년 미만 | 70% (비조정: 40%) |
| 1~2년 | 60% (비조정: 기본세율) |
| 2년 이상 | 기본세율 (6~45%) |
| 조합원입주권·분양권 | 1년 미만 70%, 그 외 60% |

- 1세대1주택 비과세: 2년 이상 보유 (조정대상지역 거주요건 추가)
- 장기보유특별공제: 3년 이상, 연 2~4% (1세대1주택 최대 80%)

### 주식 양도소득세
- 대주주 양도소득: 기본세율 (20~25%)
- 소액주주 상장주식: 비과세 (2025년 기준 금투세 폐지 확정)
- 비상장주식: 10% (중소기업) / 20% (그 외), 대주주 20~25%
