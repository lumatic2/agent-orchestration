# [KNOWLEDGE] 의학 핵심 가이드라인 (한국 기준)

## 주요 수치 정상 범위

### 혈압
| 분류 | 수축기 | 이완기 |
|---|---|---|
| 정상 | <120 | <80 |
| 주의 | 120~129 | <80 |
| 고혈압 1기 | 130~139 | 80~89 |
| 고혈압 2기 | ≥140 | ≥90 |
| 고혈압 위기 | >180 | >120 (즉시 응급실) |

### 혈당 (공복)
| 분류 | 수치 (mg/dL) |
|---|---|
| 정상 | <100 |
| 공복혈당장애 | 100~125 |
| 당뇨 진단 | ≥126 (2회 확인) |
| HbA1c 당뇨 기준 | ≥6.5% |

### 콜레스테롤
| 지표 | 정상 | 경계 | 위험 |
|---|---|---|---|
| 총콜레스테롤 | <200 | 200~239 | ≥240 |
| LDL | <100 (고위험군 <70) | 100~129 | ≥160 |
| HDL | >60 (낮으면 위험 <40) | — | — |
| 중성지방 | <150 | 150~199 | ≥200 |

### 체온·맥박·호흡수
- 정상 체온: 36.5~37.5℃ / 38℃ 이상 = 발열
- 정상 맥박: 60~100 bpm / 100↑ = 빈맥, 60↓ = 서맥
- 정상 호흡수: 12~20회/분 / 24↑ = 빈호흡 (응급)
- 정상 산소포화도: ≥95% / 90↓ = 응급

## 증상별 감별진단 주요 패턴

### 두통
| 특징 | 의심 진단 | 긴급도 |
|---|---|---|
| 갑작스러운 "생애 최악" | 지주막하출혈 | 즉시 응급 |
| 후두부 + 목 뻣뻣함 + 발열 | 뇌수막염 | 즉시 응급 |
| 편측 박동성 + 오심 + 광선공포 | 편두통 | 비응급 |
| 양측 압박감, 스트레스와 연관 | 긴장성 두통 | 비응급 |
| 새벽 두통 + 비만 + 코골이 | 수면무호흡증 | 예약 |

### 흉통
| 특징 | 의심 진단 | 긴급도 |
|---|---|---|
| 쥐어짜는 통증 + 좌측 방사 + 식은땀 | 급성 심근경색 | 즉시 응급 (119) |
| 호흡 시 악화 + 혈담 + 하지 부종 | 폐색전증 | 즉시 응급 |
| 찢어지는 흉·배통 | 대동맥 박리 | 즉시 응급 |
| 식후 악화, 제산제 반응 | 역류성 식도염 | 비응급 |
| 피부 통증 먼저 → 수포 | 대상포진 | 조기 내원 |

### 복통
| 위치 | 의심 진단 | 주의 |
|---|---|---|
| 우하복부 (맥버니 포인트) | 충수염 | 외과 응급 |
| 우상복부 + 지방식 후 악화 | 담낭염·담석 | 당일 내원 |
| 상복부 + 구토 + 음주력 | 급성 췌장염 | 응급실 |
| 하복부 + 여성 + 발열 | 자궁외임신·PID | 산부인과 응급 |
| 배꼽 주위 → 우하복부 이동 | 충수염 초기 | 응급 |

### 어지러움
| 특징 | 의심 진단 | 대처 |
|---|---|---|
| 머리 위치 변화 시 회전감 | 이석증 (BPPV) | 이비인후과, Epley 술기 |
| 귀울림 + 청력저하 + 회전감 | 메니에르병 | 이비인후과 |
| 갑작스러운 + 편측 마비·언어장애 | 뇌졸중 | 즉시 응급 |
| 일어설 때 발생, 탈수·빈혈 | 기립성 저혈압 | 내과 |

## 만성질환 자기관리 기준

### 고혈압
- 목표 혈압: 일반 <140/90, 당뇨·신장질환 동반 <130/80
- 생활습관: 나트륨 하루 2g 이하, 규칙 운동 30분/일 5회
- 복약: 임의 중단 금지 (반동성 고혈압 위험)
- 모니터링: 가정 혈압 측정 — 아침 복약 전, 저녁 취침 전

### 당뇨
- 목표 HbA1c: 일반 <7%, 고령·합병증 <8%
- 저혈당 대처: 혈당 <70mg/dL → 포도당 15g (사탕 3~4개) 섭취 → 15분 후 재측정
- 발 관리: 매일 시진, 상처 발생 시 즉시 내원
- 합병증 스크리닝: 연 1회 안저검사·신기능·신경병증 검사

### 번아웃·수면장애
- 수면위생: 취침·기상 시각 일정, 침실 스마트폰 금지, 카페인 오후 2시 이후 금지
- 번아웃 지표: 정서적 소진 + 냉소 + 효능감 저하 3가지 동시 → 정신건강의학과 상담
- 멜라토닌: 처방전 없이 구매 가능 (약국), 0.5~5mg 취침 30분 전

## 한국 건강보험 주요 급여 기준

### 의료기관 단계별 본인부담률
| 기관 | 외래 본인부담 | 비고 |
|---|---|---|
| 의원 (1차) | 30% | 동네 의원 |
| 병원 (2차) | 40% | 100~300병상 |
| 종합병원 (3차) | 50~60% | 대학병원 |
| 상급종합병원 직접 방문 | 60% | 의원 의뢰서 없을 시 비급여 가산 |

### 주요 검사 급여 여부
- 혈액검사 (일반·CBC·간기능·신기능): 급여 (증상 있을 때)
- 내시경 (위·대장): 급여 (증상 또는 국가검진)
- MRI: 급여 조건 제한적 (뇌·척추 등 특정 적응증)
- CT: 외래 50% 급여, 응급 시 급여 확대
- 국가암검진: 5대 암 (위·대장·폐·간·유방·자궁경부) 무료 또는 저비용

### 응급실 이용
- 응급실 본인부담: 경증(비응급) 방문 시 가산 적용 (진료비 외 응급의료관리료)
- 야간·휴일 진료: 의원급 의원 야간 가산 있음
- 119: 이송 비용 무료 (응급환자 기준)

## 응급 레드플래그 체크리스트

즉시 119 호출 또는 응급실:
- [ ] 갑작스러운 심한 두통 ("생애 최악")
- [ ] 흉통 + 식은땀 + 호흡곤란
- [ ] 편측 마비·안면 비대칭·언어장애 (뇌졸중 FAST)
- [ ] 의식 저하·혼수
- [ ] 산소포화도 90% 미만
- [ ] 혈압 180/120 이상 + 증상
- [ ] 심한 복통 (판자 같이 딱딱한 복부)
- [ ] 대량 출혈
- [ ] 알레르기 반응 + 호흡곤란 (아나필락시스)
