시스템 전체 지도를 읽고, 현재 상황에 맞는 안내를 제공한다.

다음 순서대로 실행해라:

1. MCP로 SYSTEM_MAP 읽기:
   `mcp__obsidian-vault__read_note("00-System/SYSTEM_MAP.md")`

2. $ARGUMENTS가 있으면 해당 키워드에 맞는 섹션만 요약해서 답해라.
   없으면 전체 구조를 간결하게 요약하고, 현재 세션 컨텍스트에서 관련 있어 보이는 섹션을 강조해라.

3. 추가로 깊이 봐야 할 파일이 있으면 포인터만 알려줘라 (직접 읽지 말고).

예시:
- `/map` → 전체 시스템 구조 요약
- `/map vault` → vault 관련 구조만
- `/map 에이전트` → 에이전트 라우팅 구조만
- `/map 길을 잃었어` → 체크리스트 안내
