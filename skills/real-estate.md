# /real-estate — 부동산 경매 분석 도구

부동산 경매 물건 분석, 수익률 계산, 권리분석 보조를 수행한다.

---

## 모드 선택

AskUserQuestion을 호출해라:
- 질문: "어떤 작업을 할까요?" (header: "모드")
- A) 분석 — 경매 물건 권리분석 + 수익률 계산
- B) 시세 — 특정 지역/아파트 시세 조사 (Gemini 위임)
- C) 공부 — 개념 설명 + 퀴즈 (vault 노트 연동)
- D) 임장 — 임장 체크리스트 생성 + 기록

---

## 모드 1: 분석 (경매 물건)

### STEP 0 — 물건 정보 수집

다음 정보를 수집해라. 없는 항목은 질문.

1. **사건번호** (예: 2026타경12345)
2. **소재지** — 주소
3. **물건 종류** — 아파트 / 빌라 / 오피스텔 / 상가 / 토지 / 기타
4. **감정가**
5. **최저매각가** (현재 회차)
6. **유찰 횟수**
7. **등기부등본 요약** — 갑구(소유권), 을구(근저당/전세권 등)
8. **임차인 현황** — 전입일, 확정일자, 보증금, 배당요구 여부
9. **특이사항** — 법정지상권, 유치권, 분묘기지권 등

### STEP 1 — 권리분석

```
## 권리분석 결과

**말소기준권리**: [근저당/가압류/담보가등기 중 최선순위]
**말소기준일**: YYYY-MM-DD

### 인수되는 권리 (낙찰자가 떠안는 것)
| 권리 | 권리자 | 설정일 | 금액 | 인수 사유 |
|---|---|---|---|---|

### 소멸되는 권리
| 권리 | 권리자 | 설정일 | 금액 |
|---|---|---|---|

### 임차인 분석
| 임차인 | 전입일 | 확정일자 | 보증금 | 대항력 | 배당순위 | 예상배당 | 인수여부 |
|---|---|---|---|---|---|---|---|

### 위험 요소
- [ ] 법정지상권 가능성
- [ ] 유치권 신고
- [ ] 분묘기지권
- [ ] 선순위 전세권
- [ ] 기타: ___

**위험도: [🟢 낮음 / 🟡 보통 / 🔴 높음]**
```

## 모드 2: 시세 조사

리서치가 필요하므로 Gemini에 위임한다:
```bash
bash ~/projects/agent-orchestration/scripts/orchestrate.sh gemini "부동산 시세 조사: [사용자 요청 내용]" real-estate-price
```

결과를 정리하여 vault `10-knowledge/real-estate/` 에 저장.

---

## 모드 3: 공부

1. vault `10-knowledge/real-estate/` 에서 관련 노트를 읽는다
2. 사용자가 요청한 개념을 설명한다
3. 이해도 확인 퀴즈 3문제를 낸다
4. 새로 정리한 내용은 vault에 업데이트한다

---

## 모드 4: 임장 체크리스트

다음 체크리스트를 생성하고, 임장 후 기록을 vault `10-knowledge/real-estate/06-임장노트.md`에 append.

```
## 임장 체크리스트: [주소]
**날짜**: YYYY-MM-DD

### 외부
- [ ] 건물 외관 상태 (균열, 누수 흔적)
- [ ] 주차장 상태
- [ ] 주변 환경 (소음, 혐오시설, 편의시설)
- [ ] 대중교통 접근성 (역 도보 ___분)
- [ ] 주변 시세 확인 (부동산 방문)

### 내부 (가능한 경우)
- [ ] 방 상태 / 곰팡이 / 누수
- [ ] 채광 / 통풍
- [ ] 수압 / 배수
- [ ] 인테리어 수리 필요 범위

### 점유 상태
- [ ] 현재 거주자 유무
- [ ] 명도 난이도 예상 (협조적/비협조적)
- [ ] 점유자와 대화 가능 여부

### 법적 확인
- [ ] 현장과 등기부 일치 여부
- [ ] 불법 증축/용도변경 여부
- [ ] 도시계획 확인 (재개발/재건축 여부)

### 총평
- 점수: ___/10
- 메모:
```

## 세율 참고 (2026 기준, 변동 가능)

### 취득세
| 구분 | 세율 |
|---|---|
| 주택 (6억 이하, 1주택) | 1% |
| 주택 (6~9억, 1주택) | 1~3% |
| 주택 (9억 초과, 1주택) | 3% |
| 다주택 (2주택, 조정지역) | 8% |
| 다주택 (3주택+) | 12% |

### 양도세 (보유기간)
| 보유기간 | 세율 |
|---|---|
| 1년 미만 | 70% |
| 1~2년 | 60% |
| 2년+ | 기본세율 (6~45%) |

> 세율은 정책에 따라 자주 변동. 실제 입찰 전 반드시 최신 세율 확인할 것.
