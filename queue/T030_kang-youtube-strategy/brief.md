# Task: 강환국 유튜브 채널 핵심 전략 분석

## 목적

그의 핵심 투자 전략 체계를 포괄적으로 정리한다.

## 채널 정보
- 채널명: 강환국
- 주요 채널 URL 후보: https://www.youtube.com/@강환국 (검색으로 확인)
- 관련 키워드: 클101, 젠포트, 퀀트킹, 올웨더, 소형주, 팩터투자, 강환국

## 분석 요청 항목

### 1. 전략 체계 전반
강환국이 운영/추천하는 전체 전략 목록:
- 각 전략명, 핵심 팩터, 리밸런싱 주기, 유니버스
- 전략 간 관계 (계층 구조, 상호 보완)

### 2. 클101 (소형주 퀀트) 상세
- 원래 정의한 팩터 5개와 비중
- F-score 조건 원문
- 마켓타이밍 조건 (MT9 등)
- 최근 업데이트 사항 (2024-2025 변경된 내용 있으면)

### 3. 젠포트 전략
- 젠포트에서 실제로 사용하는 조건식 패턴
- 소형주 외 다른 젠포트 전략 (있으면)

### 4. 올웨더 / 자산배분 전략
- 강환국식 올웨더 구성 (ETF, 비중)
- 한국형 올웨더 vs 글로벌 올웨더 차이

### 5. 최근 관점 (2024-2025)
- 한국 시장 전망 발언
- 전략 파라미터 조정 언급
- 추천 종목 성향 변화
